1. 인공지능과 에이전트는 무슨 관계일까?

'인공지능'이 사람의 머리(두뇌)와 같다면, '에이전트'는 환경을 느끼고 직접 움직이는 몸(신체나 프로그램)과 같습니다.

인공지능이 아무리 똑똑해도 생각만 하고 있다면 우리에게 도움을 줄 수 없겠죠? 인공지능이 우리 생활에 실제로 적용되려면 센서로 상황을 파악하고, 결과를 행동으로 보여주는 '지능 에이전트(Intelligent Agent)'의 형태가 되어야 합니다.

헷갈리지 마세요! (오개념 방지)

"지능 에이전트가 알아서 다 해주면, 사람은 아무것도 안 해도 되나요?"
아닙니다! 교과서에 따르면 "지능 에이전트는 인간을 위한 것이지, 인간의 개입을 전혀 없게 하려는 것은 아니다."라고 말합니다. 에이전트는 인간의 목표 달성을 돕는 '든든한 파트너' 역할을 하는 것입니다.

2. 로봇 청소기로 보는 에이전트의 3가지 특징

지능 에이전트는 일반 프로그램과 다르게 3가지 특별한 능력이 있습니다.

① 자율성

사람이 매번 리모컨으로 조종하지 않아도 됩니다.

로봇 청소기: "바닥의 먼지를 스스로 인식하고 알아서 청소하러 이동합니다."

② 협력성

다른 에이전트나 기계들과 통신하며 정보를 나눕니다.

로봇 청소기: "거실 청소기 1호와 방 청소기 2호가 서로 어디를 청소했는지 정보를 공유합니다."

③ 목표 지향성

주어진 '목표'를 달성하기 위해 최선의 방법을 찾습니다.

로봇 청소기: "'방 전체를 깨끗하게 만든다'는 목표 달성 여부가 에이전트를 평가하는 기준이 됩니다."

정리하자면! 지능 에이전트는 목적(목표 지향성)을 가지고, 스스로 판단하며(자율성), 필요할 땐 남과 소통(협력성)하는 스마트한 시스템입니다.

How it works

3. 로봇 청소기 에이전트의 머릿속

로봇 청소기가 어떻게 '자율성'과 '목표 지향성'을 가지고 움직이는지 아주 간단한 코드로 표현해 보았습니다.

단순히 버튼을 누르면 켜지는 것이 아니라, '방이 깨끗해질 때까지' (목표) 스스로 상태를 확인하고 행동하는 것을 볼 수 있습니다.

robot_agent.c 

// 지능 에이전트(로봇 청소기)의 행동 논리

void startCleaningAgent() {
    // 목표 지향성: '방이 깨끗해질 때'까지 반복합니다.
    while (checkRoomStatus() != CLEAN) {

        // 자율성 1: 스스로 배터리를 확인합니다.
        if (getBatteryLevel() < 10) {
            moveToCharger(); // 충전소로 이동
            continue;
        }

        // 자율성 2: 카메라/센서로 먼지 위치를 스스로 파악합니다.
        Location dirtLoc = findDirt();

        // 파악한 위치로 이동하여 청소합니다.
        moveTo(dirtLoc);
        cleanArea();

        // 협력성: 다른 로봇에게 "여긴 내가 청소했어"라고 알립니다.
        sendMessageToOtherRobot("구역 A 청소 완료");
    }

    print("목표 달성! 청소를 종료합니다.");
}
