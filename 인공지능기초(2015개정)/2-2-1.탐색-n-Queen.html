1단계: 막다른 길 되돌아가기

여러분, 복잡한 미로를 탐험하고 있다고 상상해 보세요. 길을 따라 걷다가 막다른 골목이 나오면 어떻게 하나요? 벽을 뚫고 갈 수는 없으니, 다시 갈림길로 되돌아가서 다른 길을 선택할 것입니다.

💡 백트래킹 (Backtracking)

인공지능도 정답을 찾아가는 '탐색' 과정에서 규칙에 어긋나 더 이상 나아갈 수 없는 상태(막다른 길)를 만나면, 이전 단계로 되돌아가서 다른 경우의 수를 탐색합니다. 이 기법을 '백트래킹'이라고 부릅니다.

핵심 기술: 가지치기 (Pruning)

어떤 길로 가봤자 정답이 없다는 것을 미리 알았다면, 그 길은 아예 시도조차 하지 않고 잘라내 버리는 것을 말합니다. 쓸데없는 나무가지를 잘라내듯, 불필요한 계산을 줄여서 탐색 속도를 엄청나게 빠르게 만들어줍니다.

2단계: N-Queen 문제 직접 풀어보기

실습 방법 및 규칙

👑 퀸(Queen)의 움직임: 체스에서 퀸은 가로, 세로, 대각선 방향으로 장애물이 없다면 끝까지 이동할 수 있는 가장 강력한 말입니다.
  • 목표: N x N 크기의 체스판에 N개의 퀸을 서로 공격할 수 없도록 안전하게 배치하는 것입니다.
  • 직접 풀기: 빈칸을 클릭해 퀸을 놓아보세요. 충돌 시 빨간색으로 변합니다.
  • AI 애니메이션: 인공지능이 '백트래킹'으로 정답을 찾는 과정을 봅니다. 속도를 조절하거나 일시정지하며 관찰해 보세요.

3단계: 코드로 보는 백트래킹 원리

인공지능이 스스로 백트래킹을 수행하도록 파이썬 언어로 작성한 코드의 핵심 구조입니다. 유망하지 않으면 포기하고(return), 이전 상태로 되돌아가는 것이 핵심입니다.

Python 
# 백트래킹을 이용한 N-Queen 탐색 함수
def solve_n_queens(row, n, state):
    # 1. 목표 달성 확인 (마지막 줄까지 퀸을 다 놓았는지)
    if row == n:
        print("정답을 찾았습니다!")
        return True

    for col in range(n):
        # 2. 가지치기: 현재 위치에 퀸을 놓는 것이 안전한지 검사
        if is_safe(row, col, state):
            state[row] = col  # 안전하다면 퀸 배치

            # 3. 다음 줄(row + 1)에 퀸을 놓기 위해 계속 탐색 시도
            if solve_n_queens(row + 1, n, state):
                return True

            # 4. 백트래킹 (되돌아가기)
            # 위에서 정답을 못 찾고 실패해서 내려왔다면,
            # 현재 놓았던 퀸을 다시 치우고 다음 칸(col)을 시도함
            state[row] = -1

    return False # 이 경로로는 정답이 없음

4단계: 오해와 진실

흔히 하는 오해

"인공지능은 컴퓨터니까 가능한 모든 경우의 수를 하나하나 다 만들어서 정답인지 확인해 보는 거 아닌가요?"

올바른 개념

아닙니다! 모든 경우의 수를 무식하게 다 해보는 것은 '브루트 포스(Brute Force)'라고 합니다.

백트래킹은 똑똑하게 탐색합니다. 퀸을 한 줄씩 놓으면서, 방금 놓은 퀸이 규칙에 어긋난다면 그 이후의 줄은 아예 검사하지 않습니다 (가지치기). 이렇게 불필요한 탐색을 중간에 포기하기 때문에 시간이 크게 절약됩니다.
