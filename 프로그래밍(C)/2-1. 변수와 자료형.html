1. 변수와 자료형

데이터를 메모리에 저장하는 변수의 개념을 익히고, 데이터 크기에 따른 효율적인 자료형 선택 방법을 배웁니다.

Think & Objective

"컴퓨터는 숫자와 문자를 내부적으로 어떻게 구분하여 저장할까요?"

  • 변수의 개념을 이해하고 적절한 자료형으로 변수를 선언할 수 있다.
  • 자료형의 특성과 메모리 크기를 이해하고 효율적인 코딩을 할 수 있다.

2. 자료형과 sizeof 연산자

자료형은 데이터를 담는 상자의 종류입니다. sizeof 연산자로 메모리 용량을 측정합니다.

65-1.c 
#include <stdio.h>
int main()
{
	printf("char: %d\n", sizeof(char));
	printf("int: %d\n", sizeof(int));
	printf("double: %d\n", sizeof(double));
	return 0;
}
65-2.c 
#include <stdio.h>
int main()
{
	char num1 = 1, num2 = 2;
	printf("%d", sizeof(num1 + num2));
	return 0;
}

5. [심화] 변수 값의 교환(Swap)

두 변수에 저장된 값을 서로 바꾸는 알고리즘입니다. 임시 변수(temp)를 사용하는 방법과 연산자를 이용하는 방법이 있습니다.

임시 변수를 이용한 교환
70-1.c 
#include <stdio.h>
int main() {
	int a = 1, b = 2;
	int temp;
	temp = a;
	a = b;
	b = temp;
	return 0;
}
연산을 이용한 교환
70-2.c 
#include <stdio.h>
int main() {
	int a = 1, b = 2;
	a = a + b;
	b = a - b;
	a = a - b;
	return 0;
}
세 변수의 순환 교환 (A)
71-1.c 
#include <stdio.h>
int main() {
	int a = 1, b = 2, c = 3;
	int temp;
	temp = a;
	a = b;
	b = c;
	c = temp;
	printf("%d %d %d", a, b, c);
	return 0;
}
세 변수의 순환 교환 (B)
71-2.c 
#include <stdio.h>
int main()
{
	int a = 1, b = 2, c = 3;
	int temp;
	temp = c;
	c = b;
	b = a;
	a = temp;
	printf("%d %d %d", a, b, c);
	return 0;
}
