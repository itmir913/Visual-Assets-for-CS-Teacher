2. 입출력 함수

컴퓨터와 사용자 사이의 소통 창구인 표준 입출력 함수를 배우고, 데이터를 원하는 형식으로 제어하는 방법을 익힙니다.

Think & Objective

"형식을 지정하여 입력과 출력을 하는 함수는 어떻게 사용할까?"

  • 입출력 함수의 기능을 이해하고 설명할 수 있다.
  • 형식을 지정하여 데이터를 입력하고 출력할 수 있다.

표준 출력 함수: printf()

printf() 함수는 "print formatted"의 줄임말로, 데이터를 특정한 형식에 맞춰 화면에 출력합니다. 단순히 글자만 찍는 것이 아니라, 변수의 값을 문장 사이에 끼워 넣을 수 있습니다.

문법 핵심 가이드

  • 1. 큰따옴표(" ") 안에 출력 내용을 적습니다.
  • 2. \n 기호로 줄바꿈을 합니다.
  • 3. %d, %f 등 서식 지정자를 사용합니다.
Step 1

단순 문자열 출력

#include <stdio.h>

int main() {
    printf("안녕하세요!\n");
    printf("C언어 공부 시작!");
    return 0;
}

자주 쓰는 서식 지정자

%d

정수 (10진수)

integer
%f

실수 (소수점)

float
%c

문자 하나

char

* %.2f는 소수점 둘째 자리까지 출력!

표준 입력 함수: scanf()

scanf() 함수는 사용자(키보드)로부터 데이터를 입력받아 변수에 저장합니다. "scan formatted"의 약자로, 입력받을 데이터의 형식을 미리 지정해줘야 합니다.

초보자가 가장 많이 하는 실수!

scanf에서 변수 이름 앞에 반드시 &(앰퍼샌드)를 붙여야 합니다!
예: scanf("%d", &age); (O)

scanf("%d", age); (X)

&는 해당 변수의 "메모리 주소"를 알려주는 기호입니다. 주소를 알려줘야 컴퓨터가 그 자리에 값을 담을 수 있어요.

Step 1

정수 하나 입력받기

#include <stdio.h>

int main() {
    int num;
    printf("숫자를 입력하고 엔터를 치세요: ");

    // 입력 대기 상태가 됩니다.
    scanf("%d", &num);

    printf("당신이 입력한 숫자는 %d입니다.\n", num);
    return 0;
}

실습 7-2
사용자 프로필 입력 및 출력
83.c 
#include <stdio.h>
int main()
{
	char name[100];
	int age;
	char bloodType;

	puts("이름을 입력하세요");
	putchar('>');
	gets(name);

	puts("나이를 입력하세요");
	putchar('>');
	scanf("%d", &age);

	puts("혈액형을 입력하세요");
	putchar('>');
	scanf(" %c", &bloodType);

	printf("%s(%d살,%c형)", name, age, bloodType);
	return 0;
}
