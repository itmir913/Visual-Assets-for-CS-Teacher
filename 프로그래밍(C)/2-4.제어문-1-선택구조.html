4. 제어문 (Control Statement), 선택 구조

프로그램의 흐름을 내 마음대로! 조건에 따라 다른 길을 가거나, 똑같은 작업을 반복하는 제어 구조를 단계적으로 마스터합니다.

학습 목표 (Objective)

  • 순차, 선택, 반복의 제어 구조를 이해하고 예제로 직접 활용할 수 있다.
  • 중첩 제어 구조(제어문 안의 제어문)를 다양한 패턴에 적용할 수 있다.

순차 구조

위에서 아래로 정해진 순서에 따라 차례대로 코드가 실행되는 가장 기본적인 흐름 구조입니다.

선택 구조

주어진 조건의 참·거짓에 따라 서로 다른 코드를 실행하는 분기 구조입니다. (if, switch)

반복 구조

특정 조건이 만족될 때까지 동일한 코드를 계속 반복 실행하여 작업을 자동화합니다. (for, while)

기초 다지기: C언어에서 '참'과 '거짓'의 비밀

본격적으로 조건문을 다루기 전에 꼭 알아야 할 중요한 규칙이 있습니다. C언어는 참(True)과 거짓(False)을 오직 '숫자'로만 구분합니다. 조건식의 결과가 0이면 거짓(False)으로 판단하고, 0이 아닌 모든 수(1, -1, 13 등)는 참(True)으로 판단하여 코드를 실행합니다.

꼭 알아야 할 비교 연산자

  • A == B A와 B는 같다 (=는 대입, ==가 비교!)
  • A != B A와 B는 다르다
  • A > B A가 B보다 크다
  • A < B A가 B보다 작다
  • A >= B A가 B보다 크거나 같다
  • A <= B A가 B보다 작거나 같다

💡 독학 포인트

다음 예제에서 if (1)이나 if (-1) 같은 코드를 보게 될 것입니다. 당황하지 마세요!

  • if (1) : 0이 아니므로 무조건 참 (실행됨)
  • if (-1) : 0이 아니므로 무조건 참 (실행됨)
  • if (0) : 0이므로 무조건 거짓 (실행 안 됨)

"C언어에서는 0만 아니면 다 참이다!"라고 기억해 두세요.

조건을 하나로 묶기: 논리 연산자

비교 연산자만으로는 복잡한 상황을 표현하기 어렵습니다. 예를 들어, "키가 130cm 이상이고, 나이가 10살 이상인가?"를 확인하려면 두 개의 조건을 동시에 검사해야 합니다. 이때 사용하는 것이 논리 연산자입니다.

&& AND (그리고)

양쪽 조건이 모두 참이어야 전체가 참이 됩니다.

if (age >= 10 && height >= 130)

|| OR (또는)

양쪽 조건 중 하나라도 참이면 전체가 참이 됩니다.

if (money >= 1000 || hasCard == 1)

! NOT (부정)

참을 거짓으로, 거짓을 참으로 반대로 뒤집습니다.

if (!isRainy) // 비가 안 오면

수학처럼 쓰면 안 돼요!

수학에서는 10 <= age <= 20 이라고 쓰지만, C언어에서는 이렇게 쓰면 컴퓨터가 오작동합니다. 반드시 논리 연산자를 사용하여 age >= 10 && age <= 20 처럼 두 조건으로 나누어 묶어주어야 합니다.

조건문 (Selection)

프로그램은 위에서 아래로 순차적으로 실행되지만, 조건문을 사용하면 상황에 따라 실행할 코드를 선택할 수 있습니다. 특정 조건이 참(1)일 때만 코드가 실행되도록 흐름을 제어합니다.

if-else 문법 상세

  • 1. if(조건식): 소괄호 안의 조건이 참이면 중괄호 { } 안의 내용을 실행합니다.
  • 2. else if: 앞의 조건이 거짓일 때, 다른 조건을 추가로 확인합니다.
  • 3. else: 제시한 모든 조건이 거짓일 때 실행되는 '나머지' 구역입니다.
Step 1

기초 조건문 (양수/음수 판별)

#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("이 숫자는 양수입니다.\n");
    } else if (num < 0) {
        printf("이 숫자는 음수입니다.\n");
    } else {
        printf("이 숫자는 0입니다.\n");
    }

    return 0;
}
Step 2

switch-case 문 (메뉴 선택)

* switch문은 여러 가지 경우(case) 중 하나를 선택할 때 if문보다 가독성이 좋습니다.

#include <stdio.h>

int main() {
    int choice = 2;

    switch (choice) {
        case 1:
            printf("콜라를 선택했습니다.\n");
            break; // break를 쓰지 않으면 다음 case까지 실행됩니다!
        case 2:
            printf("사이다를 선택했습니다.\n");
            break;
        default: // 위에 해당하지 않는 모든 경우
            printf("잘못된 선택입니다.\n");
    }
    return 0;
}

선택 구조 (단순 if 문)

단순 if 문은 조건식이 참일 때만 특정 코드를 실행하는 구조입니다. 조건이 거짓이면 if 블록 안의 코드는 생략되고 다음 코드로 넘어갑니다.

학생들이 가장 많이 하는 실수 BEST 1

수학에서의 ‘같다’는 = 이지만, C 언어의 조건문에서 ‘같다’는 반드시 ==를 사용해야 합니다. =는 대입 연산자입니다.

오류: if (score = 100) { ... } // score에 100을 대입하고, 조건식 결과가 항상 참이 되어버림
정답: if (score == 100) { ... } // score가 100인지 비교

코딩 팁: 중괄호 { }는 언제 생략할 수 있을까?

if문 아래에 실행할 코드가 딱 1줄뿐이라면 중괄호를 생략할 수 있습니다. (바로 아래 98.c 예제처럼요!)
하지만 코드가 2줄 이상인데 중괄호를 생략하면, 첫 번째 줄만 if문에 포함되고 나머지는 조건과 상관없이 무조건 실행되는 치명적인 버그가 발생합니다.
초보자라면 코드가 1줄이더라도 항상 중괄호를 쓰는 습관을 들이는 것이 좋습니다.

예제 단순 if문 (참/거짓 판별)
98.c 
#include <stdio.h>
int main()
{
    int i = 13, j = 5;
    if (-1) putchar('A');
    if (1) putchar('B');
    if (13) putchar('C');
    if (0) putchar('D');
    if (i > j)
    {
        putchar('E');
        putchar('F');
    }
    if (i < j)
        putchar('G');
    putchar('H');
    return 0;
}

선택 구조 (if-else 문)

if-else 문은 조건이 참일 때와 거짓일 때 실행되는 문장이 서로 다를 때 사용하는 선택문입니다.

예제if ~ else문 (합격/불합격)
99.c 
#include <stdio.h>
int main()
{
    int standard = 80, score = 76;
    if (standard < score)
    {
        putchar('P');
    }
    else
    {
        putchar('F');
    }
    return 0;
}

선택 구조 (다중 if 문)

다중 if 문은 if 문 안에 또 다른 if 문을 포함하는 형태로, 비교나 판단 조건을 복합적으로 사용할 수 있습니다.

예제 다중 if문 (평균 비교)
100.c 
#include <stdio.h>
int main()
{
    int standard = 173, height = 168;
    if (standard > height)
    {
        printf("평균 이하");
    }
    else if (standard == height)
    {
        printf("평균");
    }
    else
    {
        printf("평균 이상");
    }
    return 0;
}

선택 구조 심화 (중첩 if 문)

때로는 하나의 조건만으로 부족할 때가 있습니다. 예를 들어, "운전면허 필기시험에 합격한 사람 중에서, 실기시험까지 합격한 사람"을 찾으려면 어떻게 해야 할까요? 이럴 때 사용하는 것이 바로 중첩 if문(Nested if)입니다. if문 블록 안에 또 다른 if문을 넣는 방식으로, 더욱 정교한 흐름 제어가 가능합니다.

쉬운 예제 중첩 if문 (최종 합격자 판별)
nested_if.c 
#include <stdio.h>

int main() {
    int written_score = 85; // 필기 점수
    int practical_score = 70; // 실기 점수

    // 1차 조건: 필기 점수가 80점 이상인가?
    if (written_score >= 80) {
        printf("1차 필기 합격!\n");

        // 2차 조건 (중첩): 실기 점수도 60점 이상인가?
        if (practical_score >= 60) {
            printf("최종 운전면허 취득을 축하합니다!\n");
        } else {
            printf("아쉽지만 실기에서 불합격하셨습니다.\n");
        }

    } else {
        printf("필기 불합격입니다.\n");
    }

    return 0;
}

선택 구조 (switch-case 문)

비교해야 할 조건이 많을 때 다중 if 문 대신 switch-case 문을 사용하면 하나의 조건으로 여러 case 중 하나를 선택해 실행할 수 있어 코드를 더 직관적으로 만들 수 있습니다.

예제 switch문 (학점 계산)
101.c 
#include <stdio.h>
int main()
{
    int score;
    char grade;
    puts("점수를 입력하세요(1~100)");
    putchar('>');
    scanf("%d", &score);
    switch (score / 10)
    {
      case 10:
      case 9: grade = 'A'; break;
      case 8: grade = 'B'; break;
      case 7: grade = 'C'; break;
      case 6: grade = 'D'; break;
      default: grade = 'F';
    }
    printf("학점은 %c 입니다.", grade);
    return 0;
}

코드 분석: case 10 옆에는 왜 아무것도 없죠?

위 코드에서 case 10: 옆에는 코드가 비어 있고 바로 아래에 case 9:가 이어집니다. 이는 오타가 아닙니다!

switch문은 break;를 만나기 전까지 아래로 계속 떨어지며 실행되는 성질(Fall-through)이 있습니다. 따라서 100점(10)을 받든 90점대(9)를 받든, 똑같이 grade = 'A';를 실행하도록 일부러 break를 생략하여 묶어둔 것입니다.

선택 구조의 마술사: 조건(삼항) 연산자

간단한 if-else 문은 단 한 줄로 줄여서 쓸 수 있습니다. 조건식, 참일 때의 값, 거짓일 때의 값 이렇게 3가지(피연산자 3개)가 필요하다고 해서 삼항 연산자라고도 부릅니다. 코드를 매우 간결하게 만들어주어 실무에서 자주 사용됩니다.

기본 문법

조건식 ? 참일 때 값 : 거짓일 때 값 ;

💡 "조건식이 참인가요? 참이면 이것: 거짓이면 저것" 으로 소리 내어 읽으면 쉽습니다.

일반적인 if-else 문 (5줄)

int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

조건 연산자로 압축 (1줄)

int max = (a > b) ? a : b;
예제 조건 연산자 활용 (짝수/홀수 판별)
ternary.c 
#include <stdio.h>

int main() {
    int num = 15;

    // 조건: (num % 2 == 0) 이 참인가?
    // 참이면 "짝수", 거짓이면 "홀수"라는 문자열을 %s 자리에 넣습니다.
    printf("%d는(은) %s입니다.\n", num, (num % 2 == 0) ? "짝수" : "홀수");

    return 0;
}

선택 구조 핵심 요약

  • C언어의 참과 거짓 (0과 그 외) C언어는 숫자 0은 거짓(False)으로, 0이 아닌 모든 숫자(1, -1 등)는 참(True)으로 판단합니다.
  • 상황에 맞는 구문 선택 (if vs switch) 범위 조건이나 복잡한 논리는 if-else 문을, 정수나 문자 값에 따른 단순 분기는 switch 문을 사용하는 것이 효율적입니다.
  • Switch문의 Break 필수 사용 break 문이 없으면 아래쪽 case들이 연달아 실행되므로, 의도치 않은 결과를 방지하기 위해 반드시 작성해야 합니다.
