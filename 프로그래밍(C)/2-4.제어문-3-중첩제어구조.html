4. 제어문 (Control Statement), 중첩 제어구조

프로그램의 흐름을 더욱 정교하게! 제어문 안에 또 다른 제어문을 넣는 중첩 구조를 통해, 복잡한 논리와 패턴을 단계적으로 마스터합니다.

학습 목표 (Objective)

  • 중첩 반복문의 작동 원리(외부 반복문 1회 실행 시 내부 반복문이 전체 실행됨)를 완벽히 이해한다.
  • 반복 구조와 선택 구조를 결합하여 데이터를 원하는 조건에 맞게 걸러내는 논리를 익힌다.

1. 순차 구조

위에서 아래로 정해진 순서에 따라 코드가 실행됩니다.

2. 선택 구조

조건(if/switch)에 따라 서로 다른 코드를 실행합니다.

3. 반복 구조

조건이 만족될 때까지 코드(for/while)를 계속 반복합니다.

제어문 안의 제어문: 중첩 (Nesting)

이전 단원들에서 배운 선택 구조(if)와 반복 구조(for)는 단독으로 쓰일 때도 강력하지만, 서로 결합할 때 비로소 진정한 마법이 시작됩니다. 반복문 안에 또 다른 반복문이 들어가거나, 반복문 안에 선택문이 들어가는 것을 중첩 제어 구조(Nested Control Structure)라고 부릅니다.

시계열 비유: 시침과 분침

중첩 반복문을 이해하는 가장 쉬운 방법은 아날로그 시계를 떠올리는 것입니다.
바깥쪽 반복문이 '시(Hour)'라면, 안쪽 반복문은 '분(Minute)'입니다.
시침(바깥 반복문)이 1시간 움직이기 위해서는, 분침(안쪽 반복문)이 0분부터 59분까지 전부 돌아야 합니다.

한눈에 보는 중첩 반복문 실행 흐름

아래 코드는 외부 반복문(i)이 2번, 내부 반복문(j)이 3번 도는 중첩 for문의 예시입니다. 컴퓨터가 실제로 어떤 순서로 코드를 실행하는지 표를 통해 확인해 보세요.

nested_loop.c 
// 외부 반복문 (시침)
for (int i = 1; i <= 2; i++) {
    // 내부 반복문 (분침)
    for (int j = 1; j <= 3; j++) {
        printf("i=%d, j=%d\n", i, j);
    }
}

핵심 포인트: 외부 반복문이 한 번 실행될 때마다 내부 반복문은 처음부터 끝까지(새로 초기화되어) 완전히 실행됩니다. 따라서 printf()문의 총 실행 횟수는 외부 반복 횟수(2) × 내부 반복 횟수(3) = 6번이 됩니다.

외부 (i) 내부 (j) 실행 및 출력 결과
i = 1
(첫 번째 사이클)		 j = 1 i=1, j=1
j = 2 i=1, j=2
j = 3 i=1, j=3
내부 반복문 종료 (j가 3까지 도달)
i = 2
(두 번째 사이클)		 j = 1 i=2, j=1
내부 반복문 변수 j가 1로 초기화됨!
j = 2 i=2, j=2
j = 3 i=2, j=3
외부 반복문 종료 (i가 2까지 도달)

반복문과 선택문의 만남 (Filtering)

반복문이 무작정 똑같은 코드를 여러 번 실행한다면, 선택문(if)을 반복문 안에 넣으면 원하는 조건의 데이터만 골라내는 마법을 부릴 수 있습니다. 1부터 10까지 반복하면서 짝수만 화면에 출력하는 예제를 살펴보겠습니다.

filter_even.c 
#include <stdio.h>

int main() {
    // 1부터 10까지 순차적으로 반복 (총 10회)
    for (int i = 1; i <= 10; i++) {
        // 내부에서 조건문(if)을 통해 i가 짝수일 때만 출력
        if (i % 2 == 0) {
            printf("%d ", i);
        }
    }
    printf("\n");
    return 0;
}

실행 결과

2 4 6 8 10

프로그래머의 통과의례: 별 찍기

중첩 반복문을 배울 때 전 세계 모든 학생들이 거쳐 가는 예제가 있습니다. 바로 화면에 별(*)로 도형을 그리는 것입니다. 외부 반복문은 '줄(Row)'을 바꾸고, 내부 반복문은 '칸(Column)'에 별을 출력합니다.

star_triangle.c 
#include <stdio.h>

int main() {
    int n = 5; // 5줄을 출력할 예정

    for (int i = 1; i <= n; i++) {       // 줄(행)을 담당하는 외부 반복문
        for (int j = 1; j <= i; j++) {   // 별의 개수를 담당하는 내부 반복문
            printf("*"); // 줄바꿈 없이 이어서 출력
        }
        printf("\n"); // 내부 반복문이 끝나면 1줄 띄우기!
    }
    return 0;
}

출력 결과 및 분석

*
**
***
****
*****
  • i=1 내부 반복문 조건(j <= 1): 별 1개 찍고 줄바꿈
  • i=2 내부 반복문 조건(j <= 2): 별 2개 찍고 줄바꿈
  • ... 외부 반복문 변수 i가 내부 반복문의 종료 조건으로 쓰이는 마법입니다!

잠깐! 흔히 발생하는 실수들

내부와 외부의 변수명 충돌

외부 반복문에서 i를 썼는데, 내부에서도 무심코 i를 또 사용하면 논리가 완전히 엉키게 됩니다.

for (int i = 0; i < 5; i++) {
  // 안에서 또 i를 쓰면 무한루프 위험!
  for (int i = 0; i < 3; i++) { ... }
}

변수는 명확히 분리하기

관례적으로 외부부터 안쪽으로 i, j, k 순서로 변수명을 지정하여 헷갈리지 않도록 작성합니다.

for (int i = 0; i < 5; i++) {
  // 올바른 변수 j 사용
  for (int j = 0; j < 3; j++) { ... }
}
코드가 클립보드에 복사되었습니다.
