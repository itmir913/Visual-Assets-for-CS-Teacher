Chapter 2. 프로그래밍의 기초

2-1-1. 데이터 출력

파이썬이 데이터를 어떻게 구분하는지 배우고, 그 데이터를 화면에 멋지게 출력하는 방법을 마스터해 봅시다.
문법 이해와 예제 실습을 통해 출력의 기초를 다집니다.

1. 자료형(Data Type)이란?

핵심 문법 정리

컴퓨터는 숫자(정수, 실수)와 글자(문자열)를 완전히 다르게 취급해요.

  • 정수(int): 소수점이 없는 숫자
  • 문자열(str): 따옴표로 감싸진 모든 데이터

자료형 연산 실습 예제

예제 45_1 숫자와 문자열의 덧셈
45_1.py 
print(1 + "1")
Error

TypeError

숫자와 문자는 서로 더할 수 없어요!

예제 45_2 문자열의 곱셈 반복
45_2.py 
print(3 * "hello")
Result

hellohellohello

Tip

문자를 곱하면 횟수만큼 반복됩니다!

2. print() 함수와 쉼표(,)

여러 데이터 출력하기

`print()` 함수 괄호 안에 여러 데이터를 넣고 싶을 때는 쉼표(,)로 구분합니다.

  • 데이터 사이에 쉼표를 찍으면 화면에는 공백(한 칸 띄어쓰기)이 자동으로 생깁니다.
  • 숫자와 문자를 동시에 나열하여 출력할 때 유용하게 쓰입니다.

기본 출력 실습 예제

예제 47 단일 문자열 출력
47.py 
print("Hello")
Result

Hello

예제 48 여러 문자열 출력 (쉼표 사용)
48.py 
print("Hello", "World")
Result

Hello World

Tip

쉼표(,)는 자동 띄어쓰기 역할을 해요!

3. sep과 end 옵션 활용

출력 방식을 더 디테일하게 제어하고 싶을 때 sep과 end 옵션을 활용합니다.

출력 옵션 실습 예제

예제 49 구분자 변경 (sep)

데이터 사이를 공백 대신 특정 문자로 채웁니다.

49.py 
print(13, 45, 29, sep=":")
Result

13:45:29

예제 50 기본 동작 (줄바꿈)

print() 함수는 호출될 때마다 기본적으로 자동으로 줄을 바꿉니다.

50.py 
print("Hello")
print("World")
Result

Hello
World

예제 51 줄 바꿈 조정 (end)

출력 후 줄을 바꾸는 대신, 다음 출력을 지정한 문자로 이어붙입니다.

51.py 
print("A", end=",")
print("B", end=",")
print("C")
Result

A,B,C

Tip

end=""를 사용하면 줄바꿈 없이 문자를 계속해서 연결할 수 있습니다.

데이터 출력 핵심 요약

print() (기본 출력)
sep=" " (구분자 변경)
end="" (줄바꿈 방지)

"올바른 자료형과 출력 옵션을 이해하면, 원하는 포맷으로 데이터를 자유롭게 화면에 표시할 수 있습니다."
