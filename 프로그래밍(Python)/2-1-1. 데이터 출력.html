2-1-1. 데이터 출력
세상의 모든 정보를 화면에!

프로그램이 계산한 결과를 우리가 볼 수 있게 꺼내주는 과정인 '출력'과, 그 데이터가 담긴 '자료형'의 원리를 마스터해 봅시다.

1. 자료형(Data Type)의 이해

컴퓨터는 모든 데이터를 0과 1로 저장하지만, 사람이 다루는 데이터는 종류가 다양합니다. 자료형이란 컴퓨터가 데이터를 어떻게 해석하고 처리해야 할지 알려주는 **'데이터의 이름표'**와 같습니다.

정수형 (int) Integer

소수점이 없는 숫자입니다. 양수, 0, 음수가 포함됩니다.

예: 10, 0, -5

실수형 (float) Floating-point

소수점이 포함된 숫자입니다. 정밀한 계산이 필요할 때 사용합니다.

예: 3.14, -0.01, 2.0

문자열 (str) String

따옴표로 감싼 글자들의 집합입니다. 숫자도 따옴표 안에 있으면 글자가 됩니다.

예: "안녕", 'Python', "123"

논리형 (bool) Boolean

참(True)과 거짓(False) 두 가지 상태만 가집니다.

예: True, False (첫 글자 대문자 주의!)

학생들이 가장 많이 틀리는 포인트!

123은 숫자(정수)이지만, "123"은 글자(문자열)입니다. 정수 123은 더하기 연산이 가능하지만, 문자열 "123"은 그냥 글자 취급을 받으므로 수학적 계산 방식이 달라집니다.

2. print() 함수 완전 정복

교과서의 모든 예제를 완벽하게 분석합니다.

Step 01

기본 출력: Hello World

가장 기본적인 형태입니다. 괄호 안에 출력하고 싶은 값을 넣습니다. 문자열은 반드시 따옴표를 사용하세요.

💡 학습 팁

파이썬은 명령어가 끝나고 자동으로 줄을 바꿉니다. (Enter 효과)

print("Hello World")
print("World")
Console Output 
Hello World
World
Step 02

쉼표(,)의 마법

여러 데이터를 한 번에 출력하고 싶을 때 쉼표를 사용합니다. 파이썬은 쉼표를 만나면 자동으로 한 칸 공백(Space)을 넣어줍니다.

"안녕", "파이썬" → 안녕 파이썬

print("Hello", "Python", "World")
Console Output 
Hello Python World
Advanced 01

sep 옵션: 내 맘대로 구분자

쉼표가 만드는 기본 공백 대신 다른 문자를 넣고 싶을 때 sep(Separator) 옵션을 사용합니다.

사용처: 날짜(.), 전화번호(-), 시간(:) 등

print("3", "01", "004", sep=".")
print("010", "1234", "5678", sep="-")
Console Output 
3.01.004
010-1234-5678
Advanced 02

end 옵션: 줄 바꿈 금지!

파이썬은 기본적으로 출력이 끝나면 줄을 바꿉니다(\n). 이를 방지하고 싶다면 end 옵션을 사용하세요. 

print("Hello", end=" ")
print("World")
Console Output 
Hello World

(줄이 바뀌지 않고 한 줄에 출력됨)
