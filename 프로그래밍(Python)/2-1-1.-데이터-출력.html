Chapter 2. 프로그래밍의 기초

2-1-1. 데이터 출력

파이썬이 데이터를 어떻게 구분하는지 배우고, 그 데이터를 화면에 멋지게 출력하는 방법을 마스터해 봅시다.

Think & Objective

"여러 가지 데이터를 컴퓨터로 다루기 위해서는 어떻게 입력하고 출력해야 할까?"

  • 주어진 값을 자료형에 따라 정확하게 표현할 수 있다.
  • 주어진 데이터를 한 줄 또는 여러 줄로 출력할 수 있다.

1. 자료형(Data Type)의 개념과 특징

컴퓨터 내부에서 데이터는 종류와 특성에 따라 저장 및 처리 방식이 다릅니다. 이를 자료형(data type)이라고 합니다.

정수 (Integer)

소수점이 없는 딱 떨어지는 숫자입니다. (예: -1, 0, 100)

"컴퓨터 내부에는 2진수 형태로 저장됩니다."

실수 (Floating-point)

소수점을 포함한 숫자입니다. (예: 3.14, 0.5, -2.0)

"정수와는 다른 특별한 방식으로 저장됩니다."

문자열 (String)

큰따옴표(" ")나 작은따옴표(' ')로 감싼 데이터입니다. (예: "안녕", '123')

논릿값 (Boolean)

참(True) 또는 거짓(False) 중 하나의 값을 가집니다.

파이썬의 연산 규칙 (교과서 핵심)

  • 정수 + 문자열: 자료형이 달라 연산이 불가능하며 오류가 발생합니다.
  • 문자열 * 정수: 해당 문자열을 정수 횟수만큼 반복하여 출력합니다.

자료형 연산 실습 예제

예제 45_1 숫자와 문자열의 덧셈
45-1.py 
print(1 + "1")
Error

TypeError: unsupported operand type(s)

숫자와 문자는 서로 더할 수 없어요!

예제 45_2 문자열의 곱셈 반복
45-2.py 
print(3 * "hello")
Result

hellohellohello

Tip

문자를 곱하면 횟수만큼 반복됩니다!

2. 자료형 변환 함수

함수명 기능 설명 예시
int(x) x를 정수 자료형으로 변환 int("10") → 10
float(x) x를 실수 자료형으로 변환 float("3.14") → 3.14
str(x) x를 문자열 자료형으로 변환 str(100) → "100"
chr(x) / ord(x) 유니코드 정수와 문자 간 변환 chr(65) → 'A'

3. 화면 출력 (print) 마스터하기

01

기본 출력과 쉼표(,) 활용

print() 함수 내에서 값을 쉼표로 나열하면 공백(스페이스)을 사이에 두고 순서대로 출력됩니다.

예제 47 기본 데이터 출력
47.py 
print("Hello")
Result

Hello

예제 48 쉼표를 이용한 나열
48.py 
print("Hello", "Python", "World")
Result

Hello Python World

02

구분자 설정 (sep 옵션)

데이터 사이에 공백 대신 특정 문자를 넣고 싶을 때 sep 옵션을 사용합니다.

예제 49 구분자 지정 출력
49.py 
print("3", "01", "004", sep=".")
Result

3.01.004

03

줄 바꿈 제어 (end 옵션)

기본적으로 print()는 자동 줄바꿈을 수행합니다. 줄을 바꾸지 않고 이어 붙이려면 end 옵션을 사용합니다.

예제 50 기본 줄 바꿈 동작
50.py 
print("Hello")
print("World")
Result

Hello
World

예제 51 줄 바꿈 없이 출력
51.py 
print("Hello", end=", ")
print("Python")
Result

Hello, Python
