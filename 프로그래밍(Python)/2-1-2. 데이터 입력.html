Chapter 2. 프로그래밍의 기초

2-1-2. 데이터 입력
컴퓨터와 대화하는 법

사용자의 명령을 받아들이는 '입력'과 데이터를 소중히 담아두는 '변수'의 개념을 익혀 봅시다.

1. 변수(Variable)의 이해

프로그래밍에서 변수란 데이터를 저장하기 위해 이름을 붙인 **'메모리 상의 상자'**입니다. 한 번 저장된 값은 나중에 그 이름을 불러 언제든지 다시 사용할 수 있습니다.

변수 이름 짓기 규칙

  • 영문자, 숫자, 언더바(_)만 사용 가능
  • 숫자로 시작할 수 없음
  • 대문자와 소문자를 엄격히 구분함
  • 파이썬 예약어(if, for, print 등)는 사용 불가

대입 연산자 (=)

수학에서 =은 '같다'는 뜻이지만, 프로그래밍에서는 **"오른쪽 값을 왼쪽 변수에 집어넣으라"**는 뜻입니다.

score = 100 # 100을 score 상자에 저장!

2. input() 함수와 자료형 변환

컴퓨터에게 정보를 전달하는 가장 핵심적인 도구입니다.

Basic 01

문자열 입력받기

input() 함수는 사용자가 키보드로 친 내용을 가져옵니다. 괄호 안에 질문을 써두면 사용자에게 보여줍니다.

🚨 주의사항

input()으로 받은 데이터는 무조건 '문자열'로 들어옵니다.

name = input("이름을 입력하세요: ")
print(name, "님 반갑습니다!")
Console Interaction

이름을 입력하세요: 지민

지민 님 반갑습니다!

Basic 02

정수/실수로 변환하기

숫자를 입력받아 계산을 하려면 반드시 int()나 float()로 감싸줘야 합니다.

📌 활용 공식

정수형 = int(input())

실수형 = float(input())

age = int(input("나이를 입력하세요: "))
print("내년에는", age + 1, "살이 되네요!")
Console Interaction

나이를 입력하세요: 17

내년에는 18 살이 되네요!

Advanced 01

공백으로 구분해 입력받기

split() 함수를 사용하면 한 번에 여러 개의 데이터를 나누어 입력받을 수 있습니다. 

# 시, 분, 초를 공백으로 나누어 입력
h, m, s = input("시 분 초 입력: ").split()
print(h, "시", m, "분", s, "초")
Console Interaction

시 분 초 입력: 12 30 55

12 시 30 분 55 초

Advanced 02

여러 숫자 입력받기 마스터

여러 숫자를 한 줄에 입력받아 즉시 숫자로 바꾸고 싶을 때는 map(int, ...) 기법을 사용합니다. 

# 여러 숫자를 쉼표로 구분해 입력받고 정수로 변환
nums = list(map(int, input("숫자 3개(쉼표 구분): ").split(',')))
print("첫 번째 숫자:", nums[0])
print("합계:", sum(nums))
Console Interaction

숫자 3개(쉼표 구분): 10,20,30

합계: 60
