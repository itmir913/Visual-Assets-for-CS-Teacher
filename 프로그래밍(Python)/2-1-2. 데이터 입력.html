Chapter 2-1-2

데이터 입력 및 변수 활용

사용자로부터 값을 입력받아 변수에 저장하고, 저장된 데이터를 자유자재로 다루는 방법을 마스터해 봅시다.

1. 변수(Variable)의 기본

변수란?

변수는 데이터를 저장하기 위해 이름을 붙인 메모리 공간입니다.

  • 할당 연산자(=): 값을 변수에 저장할 때 사용합니다.
  • 동적 타이핑: 파이썬은 변수에 들어있는 값에 따라 자료형이 결정됩니다.

다양한 실습 예제

예제 53 변수 선언과 여러 번 출력하기
53.py 
s = "Hello"
print(s)
print(s)
print(s)
예제 54 변수 값 덮어쓰기
54.py 
n = 0
print(n)
n = 1
print(n)
print(n)
Tip

변수에 새로운 값을 넣으면 이전 값은 사라지고 덮어씌워집니다.

예제 55 변수 여러 개 활용하여 출력 순서 바꾸기
55.py 
s1 = "Hello"
s2 = "Python"
s3 = "World"
print(s1, s2, s3)
print(s2, s3, s1)
print(s3, s1, s2)

2. 리스트와 변수 범위

예제 56_2 리스트 인덱싱
56_2.py 
d = [1, 2, 3, 4, 5]
print(d[4], d[3], d[2], d[1], d[0])
Result

5 4 3 2 1

예제 57 전역 변수(Global) 활용
57.py 
s = 0
def f(n):
  global s
  k = 0
  while k < n:
    k = k + 1
    s = s + k
f(3)
print(s)

3. input() 함수와 데이터 입력

예제 58 한 줄 입력받아 출력하기
58.py 
s = input()
print(s)
예제 59 여러 줄 연속으로 입력받기
59.py 
s1 = input()
s2 = input()
print(s1)
print(s2)
Caution

input()의 결과는 항상 '문자열'입니다.

숫자로 계산하고 싶다면 나중에 배울 int() 변환이 필요합니다.

4. 효율적인 입력 처리 기법

예제 60 공백으로 분리하기 (여러 변수에 저장)
60.py 
s1, s2 = input().split()
print(s1)
print(s2)
예제 61 공백으로 분리하기 (리스트 인덱싱)
61.py 
s = input().split()
print(s[0])
print(s[1])
print(s[2])
예제 63 특정 구분자 활용 (예: 시각 분리)
63.py 
h, m, s = input().split(':')
print(h)
print(m)
print(s)
Tip

split(':')와 같이 괄호 안에 문자를 넣으면 해당 문자를 기준으로 데이터를 자릅니다.

예제 62 숫자 리스트로 즉시 변환 (map)
62.py 
d = list(map(int, input().split()))
print(d[0], d[1], d[2], d[3], d[4])
예제 64 콤마(,) 구분자를 숫자 리스트로 변환
64.py 
d = list(map(int, input().split(',')))
print(d[0], d[1], d[2], d[3], d[4], sep='\n')
Advanced

map(int, ...)는 모든 입력값을 한꺼번에 정수로 바꾸는 마법 같은 코드입니다!

데이터 입력 핵심 요약

input() (기본 입력)
split() (문자 분리)
map(int, ...) (정수 일괄 변환)
