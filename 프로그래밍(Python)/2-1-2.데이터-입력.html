Chapter 2. 프로그래밍의 기초

2-1-2. 데이터 입력 및 변수 활용

사용자로부터 값을 입력받아 변수에 저장하고, 저장된 데이터를 자유자재로 다루는 방법을 마스터해 봅시다.

Think & Objective

"여러 가지 데이터를 컴퓨터로 다루기 위해서는 어떻게 입력하고 출력해야 할까?"

  • 변수를 사용해 데이터를 저장할 수 있다.
  • 원하는 형식으로 데이터를 출력할 수 있다.

1. 변수(Variable)의 기본

변수란?

변수란 데이터가 저장된 위치를 가리키는 특별한 이름(메모리 공간)입니다. 여러 가지 데이터를 저장한 후 이를 불러 사용할 때 변수(variable)를 만들어 사용할 수 있습니다.

변수를 활용하면 여러 가지 이름으로 데이터를 저장할 수 있고, 필요할 때마다 편리하게 불러 재사용이 가능합니다.

  • 할당 연산자(=): 값을 변수에 저장할 때 사용합니다.
  • 동적 타이핑: 파이썬은 변수에 들어있는 값에 따라 자료형이 결정됩니다.

▲ 변수 이름으로 사용할 수 없는 것 (주의!)

프로그래밍 언어에서 이미 문법적인 용도로 사용 중인 예약어(if, for, while 등)나 내장 함수 이름(print, input 등)은 변수 이름으로 사용할 수 없습니다.

다양한 실습 예제

예제 변수 선언과 여러 번 출력하기
53.py 
s = "Hello"
print(s)
print(s)
print(s)
예제 변수 값 덮어쓰기
54.py 
n = 0
print(n)
n = 1
print(n)
print(n)
Tip

프로그램 실행 도중 새로운 값을 변수에 저장하면 새로운 값으로 덮어쓰기(overwrite) 되고, 다른 값으로 바꾸기 전까지 그대로 유지됩니다.

예제 변수 여러 개 활용하여 출력 순서 바꾸기
55.py 
s1 = "Hello"
s2 = "Python"
s3 = "World"
print(s1, s2, s3)
print(s2, s3, s1)
print(s3, s1, s2)
Tip

여러 개의 값을 저장했다가 사용해야 할 경우, 위와 같이 필요한 개수만큼 여러 개의 변수를 만들어 사용 가능합니다.

2. 리스트와 변수 범위

리스트(List)란?

여러 개의 값을 하나의 리스트로 만들어 변수에 저장하였다가 사용해야 할 경우 대괄호 [ ] 로 데이터를 나열해 리스트 변수로 저장합니다.

📌 리스트 요소 접근 방법

  • 리스트 변수에 저장된 값을 사용할 경우 대괄호 [ ] + 참조 번호를 함께 사용합니다.
  • 중요: 리스트의 첫 번째 데이터 번호는 항상 0입니다. (1이 아닙니다!)
예제 리스트 인덱싱
56_2.py 
d = [1, 2, 3, 4, 5]
print(d[4], d[3], d[2], d[1], d[0])
Result

5 4 3 2 1

변수의 종류 (Scope)

전역 변수 (global variable)

프로그램의 모든 위치에서 참조해 사용할 수 있는 변수입니다.

지역 변수 (local variable)

함수 안과 같이 특정 범위 내에서만 참조해 사용할 수 있는 변수입니다.

매개 변수 (parameter)

함수 실행에 필요한 값을 함수 안으로 전달하기 위해 사용하는 변수입니다.

예제 전역 변수(Global) 활용
57.py 
s = 0
def f(n):
  global s
  k = 0
  while k < n:
    k = k + 1
    s = s + k
f(3)
print(s)

3. input() 함수와 데이터 입력

input() 함수란?

input() 함수를 사용하면 키보드를 통해 입력한 문자열을 변수에 바로 저장할 수 있습니다. 프로그램은 사용자가 값을 입력할 때까지 일시 정지합니다.

예제 한 줄 입력받아 출력하기
58.py 
s = input()
print(s)
예제 여러 줄 연속으로 입력받기
59.py 
s1 = input()
s2 = input()
print(s1)
print(s2)
Caution

input()의 결과는 항상 '문자열'입니다.

숫자로 계산하고 싶다면 나중에 배울 int() 변환이 필요합니다.

4. 효율적인 입력 처리 기법

자료형 변환의 중요성

프로그래밍 언어에서는 데이터의 종류에 따라 정확한 자료형을 사용해야 합니다. 문자열 그대로는 수학적 계산이 불가능하므로, 숫자로 사용하려면 알맞은 형태(int, float 등)로 변환하는 과정이 꼭 필요합니다.

예제 공백으로 분리하기 (여러 변수에 저장)
60.py 
s1, s2 = input().split()
print(s1)
print(s2)
예제 공백으로 분리하기 (리스트 인덱싱)
61.py 
s = input().split()
print(s[0])
print(s[1])
print(s[2])
예제 특정 구분자 활용 (예: 시각 분리)
63.py 
h, m, s = input().split(':')
print(h)
print(m)
print(s)
Tip

split(':')와 같이 괄호 안에 문자를 넣으면 해당 문자를 기준으로 데이터를 자릅니다.

예제 숫자 리스트로 즉시 변환 (map)
62.py 
d = list(map(int, input().split()))
print(d[0], d[1], d[2], d[3], d[4])
예제 콤마(,) 구분자를 숫자 리스트로 변환
64.py 
d = list(map(int, input().split(',')))
print(d[0], d[1], d[2], d[3], d[4], sep='\n')
Advanced

map(int, ...)는 모든 입력값을 한꺼번에 정수로 바꾸는 마법 같은 코드입니다!
