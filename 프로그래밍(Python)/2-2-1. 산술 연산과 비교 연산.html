Chapter 2. 프로그래밍의 기초

2-2-1. 산술 연산과 비교 연산
데이터를 요리하는 마법의 기호

컴퓨터에 저장된 데이터를 계산하고(산술), 크기를 비교하여(비교) 논리적인 결과를 만들어내는 방법을 배워봅시다.

1. 산술 연산(Arithmetic)

연산(Operation)이란 미리 정해 놓은 규칙에 따라 데이터를 계산하거나 처리하는 과정입니다. 파이썬은 사칙연산 외에도 프로그래밍에서 유용하게 쓰이는 특별한 연산자들을 제공합니다.

기호 의미 예시 결과
+더하기5 + 27
-빼기5 - 23
*곱하기5 * 210
/나누기5 / 22.5
//몫 (정수 나누기)5 // 22
%나머지5 % 21
**거듭제곱2 ** 38

% 연산자는 언제 쓸까요?

나머지 연산자(%)는 정말 중요합니다! 짝수인지 홀수인지 판별할 때(2로 나눈 나머지 확인), 혹은 특정 주기로 반복되는 작업을 만들 때(예: 요일 계산) 아주 유용하게 사용됩니다.

2. 비교 연산(Comparison)

비교 연산자는 두 값의 크기를 비교합니다. 이 연산의 결과는 항상 **True(참)** 아니면 **False(거짓)**로 나옵니다. 이것을 우리는 불(Bool) 자료형이라고 불렀던 것, 기억하시죠?

# 관계 연산자
10 > 5 # True
10 < 5 # False
10 >= 10 # True
# 일치 연산자
10 == 10 # True (같다)
10 != 5 # True (다르다)
10 == 5 # False

헷갈리지 마세요! = vs ==

a = 10

대입 연산자: a라는 상자에 10을 **넣는다**는 뜻

a == 10

비교 연산자: a와 10이 **같은가?**라고 묻는 것

3. 실전 코딩 예제

배운 연산자들을 실제로 어떻게 사용하는지 확인해 봅시다.

Ex 01

사과 나누어 주기

사과 7개를 3명에게 똑같이 나누어 줄 때, 한 명이 받는 사과 수와 남는 사과 수를 구해 봅시다. 

apple = 7
people = 3

print("한 명당 사과:", apple // people)
print("남은 사과:", apple % people)
Output

한 명당 사과: 2

남은 사과: 1

Ex 02

점수 확인하기

입력받은 점수가 합격 기준(80점) 이상인지 확인하는 프로그램을 만들어 봅시다. 

score = int(input("점수 입력: "))
is_pass = (score >= 80)

print("합격입니까?", is_pass)
Interaction

점수 입력: 85

합격입니까? True

핵심 요약

나누기 삼형제

/ 는 실수 나누기
// 는 몫만
% 는 나머지만

비교의 결과

크다, 작다, 같다의 결과는 항상 True 또는 False이다.

주의할 기호

같다는 ==
다르다는 !=
제곱은 **
