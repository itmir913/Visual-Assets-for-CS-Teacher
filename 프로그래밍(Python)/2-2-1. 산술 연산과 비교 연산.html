Chapter 2-2-1

산술 연산과 비교 연산

파이썬을 사용하여 수학적인 계산을 수행하고, 값의 크기를 비교하여 참/거짓을 판단하는 핵심 문법을 배워봅시다.

1. 산술 연산자

산술 연산자는 수학적 계산을 위해 사용되는 연산자입니다.

연산자 의미 예시 결과
+덧셈5 + 27
-뺄셈5 - 23
*곱셈5 * 210
/나눗셈5 / 22.5
//몫5 // 22
%나머지5 % 21
**거듭제곱5 ** 225

산술 연산 실습 예제

예제 70 기본 연산
70.py 
print(123+456)
Result

579

예제 71 입력받아 나누기
71.py 
a = input()
b = input()
a = int(a)
b = int(b)
print(a/b)
Note

/ 연산의 결과는 항상 실수형(float)입니다.

문자열에도 산술 연산자를 쓸 수 있습니다

예제 72 문자열 연결
72.py 
s1, s2 = input().split()
print(s1+s2)
설명

+ 연산자는 두 문자열을 하나로 이어줍니다.

예제 73 문자열 반복
73.py 
s = input()
print(s*3)
설명

* 연산자는 문자열을 정수만큼 반복합니다.

2. 비교 연산자

비교 연산자는 두 값의 크기를 비교하거나 일치 여부를 판단할 때 사용하며, 결과는 언제나 참(True) 또는 거짓(False)인 불(bool) 자료형으로 나옵니다.

연산자 의미 예시 결과
==같다5 == 2False
!=다르다5 != 2True
>크다5 > 2True
<작다5 < 2False
>=크거나 같다5 >= 5True
<=작거나 같다5 <= 2False

비교 연산 실습 예제

예제 75 크기 비교
75.py 
print(123<456)
Result

True

예제 76 일치 비교
76.py 
a = input()
b = input()
a = int(a)
b = int(b)
print(a==b)
설명

변수 a와 b에 저장된 값이 같으면 True, 다르면 False를 출력합니다.

예제 77 실수 크기 비교
77.py 
a = input()
b = input()
a = float(a)
b = float(b)
print(a<=b)
Note

실수(float) 데이터도 비교 연산이 가능합니다.

예제 78 문자열 일치 비교
78.py 
s1, s2 = input().split()
print(s1==s2)
설명

문자열의 내용이 완전히 똑같은지 확인합니다.

학습 정리

나머지 연산의 활용

나머지 연산(%)은 짝수/홀수 판별이나 배수를 확인할 때 매우 중요하게 쓰입니다.

주의사항

같다는 기호는 == (두 개)입니다. = (한 개)는 변수에 값을 대입할 때만 씁니다.
