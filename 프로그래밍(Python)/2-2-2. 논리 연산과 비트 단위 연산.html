Chapter 2. 프로그래밍의 기초

2-2-2. 논리 연산과 비트 단위 연산
컴퓨터처럼 사고하는 법

여러 조건을 조합하는 논리 연산과 컴퓨터의 가장 밑바닥 언어인 비트 연산을 배워봅시다.

1. 논리 연산(Logical)

논리 연산은 참(True)과 거짓(False) 값을 가지고 새로운 결과를 만들어냅니다. 복잡한 현실의 조건(예: 성적이 80점 이상이면서 출석률이 90% 이상인 경우)을 코드로 표현할 때 필수적입니다.

not (논리 부정)

청개구리 연산자: 반대로 뒤집기

not True → False

not False → True

and (논리곱)

깐깐한 연산자: 모두 참이어야 참

True and True → True

True and False → False

or (논리합)

너그러운 연산자: 하나만 참이어야 참

True or False → True

False or False → False

선생님의 Tip: 단락 평가(Short-Circuit Evaluation)

파이썬은 효율적입니다! A and B에서 A가 거짓이면 B는 보지도 않고 바로 False를 뱉습니다. 반대로 A or B에서 A가 참이면 뒤는 보지도 않고 True를 뱉죠. 이를 이용해 오류를 방지하는 코드를 짜기도 합니다.

2. 비트 단위 연산(Bitwise)

컴퓨터 내부에서는 모든 데이터가 0과 1(비트)로 저장됩니다. 비트 연산은 정수를 2진수로 풀어서 각 자리수끼리 계산하는 아주 세밀한 연산입니다.

기호 의미 설명
&Bitwise AND둘 다 1일 때만 1
|Bitwise OR둘 중 하나라도 1이면 1
^Bitwise XOR서로 다를 때만 1 (배타적 논리합)
~Bitwise NOT0은 1로, 1은 0으로 반전 (보수)
<<, >>Shift비트를 왼쪽/오른쪽으로 밀기

비트 연산의 과정 (예: 5 & 3)

# 5를 2진수로: 00000101

# 3을 2진수로: 00000011

  0101 (5)

& 0011 (3)

----------

  0001 (결과: 1)

3. 실전 코딩 예제

논리와 비트 연산을 조합해 실생활 문제를 해결해 봅시다.

Ex 01

놀이공원 입장 제한

나이가 12세 이상이면서 키가 140cm 이상인 경우에만 '탑승 가능'을 출력해 보세요. 

age = 15
height = 145

can_ride = (age >= 12) and (height >= 140)
print("탑승 가능 여부:", can_ride)
Output

탑승 가능 여부: True

Ex 02

2배속 곱셈 기법

비트를 왼쪽으로 밀면 2의 거듭제곱만큼 곱해지는 효과가 있습니다. 시프트 연산자를 사용해 봅시다. 

n = 10
print(n << 1)  # 10 * 2 = 20
print(n << 2)  # 10 * 4 = 40
print(n >> 1)  # 10 // 2 = 5
Result

20
40
5

학생들이 가장 많이 틀리는 부분!

1. 'and'와 '&'를 혼동하지 마세요

and는 전체 값이 참/거짓일 때 사용하는 논리 연산이고, &는 각 숫자의 비트끼리 계산하는 비트 연산입니다. 결과값이 아예 다를 수 있어요!

2. ~ (NOT) 연산의 비밀

~x의 결과는 -(x+1)입니다. 비트가 반전되면서 컴퓨터의 음수 표현법(2의 보수) 때문에 이런 결과가 나오니 당황하지 마세요.

논리 연산 핵심

  • • and: 둘 다 True여야 True
  • • or: 하나만 True여야 True
  • • not: 결과를 반대로 뒤집음

비트 연산 핵심

  • • &, |, ^: 비트 단위 논리 계산
  • • <<, >>: 비트 밀기 (곱셈/나눗셈 효과)
