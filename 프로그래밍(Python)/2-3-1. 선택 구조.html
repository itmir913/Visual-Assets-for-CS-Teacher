Chapter 2. 프로그래밍의 기초

2-3-1. 선택 구조
프로그램에 지능을 불어넣다

단순히 위에서 아래로 실행되는 코드를 넘어, 상황에 따라 다른 길을 선택하는 '제어 구조'의 핵심을 배워봅시다.

1. 선택 구조란 무엇인가?

순차 구조 (Sequential)

작성된 명령어가 위에서 아래로 순서대로 하나씩 실행되는 기본 구조입니다.

선택 구조 (Selection)

주어진 조건이 참(True)인지 거짓(False)인지에 따라 실행할 명령을 결정하는 구조입니다.

2. if 문: "만약 ~라면?"

1 기본 문법 (if)

PYTHON SYNTAX
if 조건식:
    # 조건이 참일 때 실행할 코드
    실행문1

⚠️ 콜론(:)을 잊지 마세요!

조건식 뒤에는 반드시 콜론을 찍어야 합니다. "이제부터 실행할 블록이 시작된다"는 신호입니다.

⚠️ 들여쓰기가 생명입니다!

if문에 속한 코드는 반드시 동일한 간격(보통 공백 4칸)으로 들여써야 합니다.

3. if ~ else 문: "아니면 말고"

조건이 참일 때와 거짓일 때, 각각 서로 다른 두 갈래 길을 만들고 싶을 때 사용합니다. 

if 조건식:
    참일 때 실행할 문장
else:
    거짓일 때 실행할 문장
실전 예제: 합격/불합격 판별
score = int(input("점수 입력: "))

if score >= 60:
    print("합격입니다!")
else:
    print("불합격입니다. 다음 기회에..")

4. 다중 선택 (중첩 & elif)

검사해야 할 조건이 여러 개일 때 유용합니다. PPT의 중첩 선택 구조를 더 효율적으로 표현하는 파이썬의 비밀 병기 elif를 배워봅시다.

방식 1: 중첩(Nested) if 
if n >= 90:
    print("A")
else:
    if n >= 80:
        print("B")
    else:
        print("C")

계속 오른쪽으로 밀려나서 읽기 힘들어져요!

방식 2: elif (권장) 
if n >= 90:
    print("A")
elif n >= 80:
    print("B")
else:
    print("C")

수직으로 정렬되어 훨씬 깔끔하고 보기 좋습니다.

시험에 꼭 나오는 오답 유형!

비교 연산자 '=='와 대입 연산자 '=' 혼동

if a = 10: (X) → if a == 10: (O)

조건식에서는 반드시 '같다'는 의미의 ==를 써야 합니다.

IndentationError: 들여쓰기 오류

파이썬은 들여쓰기 한 칸 차이로 코드가 실행되느냐 에러가 나느냐가 결정됩니다. 탭(Tab)과 스페이스를 섞어 쓰지 마세요!
