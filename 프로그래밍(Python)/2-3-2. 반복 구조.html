단순 반복 업무를 컴퓨터에게 맡기는 방법, for문과 while문을 완벽하게 마스터해 봅시다.

1. 왜 반복문이 필요한가?

화면에 "안녕하세요"를 100번 출력해야 한다면? print문을 100번 쓰는 대신, 반복 구조를 사용하면 단 2줄의 코드로 해결할 수 있습니다. 반복문은 프로그램의 효율성을 극대화하는 핵심 요소입니다.

while 문

"조건이 참인 동안 계속해!"

횟수가 정해지지 않았을 때 유리

for 문

"정해진 범위만큼 반복해!"

횟수가 정해져 있거나 리스트를 다룰 때 유리

2. while 문: 조건 중심 반복

while문은 조건식이 True(참)인 동안 블록 내부의 명령을 무한히 반복합니다. 조건이 False(거짓)가 되는 순간 탈출합니다. 

i = 1  # 초기식
while i <= 5: # 조건식
    print(i)
    i = i + 1  # 증감식 (매우 중요!)

무한 루프(Infinite Loop) 주의!

위 코드에서 i = i + 1(증감식)을 빼먹으면 어떻게 될까요? i는 영원히 1이어서 조건이 항상 참이 되고, 프로그램은 멈추지 않고 계속 실행됩니다. 반드시 조건을 거짓으로 만들 장치를 넣으세요!

3. for 문: 범위 중심 반복

파이썬에서 가장 사랑받는 반복문입니다. 리스트나 range() 함수로 만든 수열에서 하나씩 값을 꺼내 사용합니다.

Case 01: range() 사용

for i in range(1, 6):
    print(i)
# 1부터 5까지 출력 (6미만)

Case 02: 리스트 사용

fruits = ["사과", "배", "귤"]
for f in fruits:
    print(f)

range(start, stop, step) 마스터하기

  • • range(5): 0, 1, 2, 3, 4 (0부터 5미만까지)
  • • range(1, 5): 1, 2, 3, 4 (1부터 5미만까지)
  • • range(1, 10, 2): 1, 3, 5, 7, 9 (2씩 건너뛰며!)
  • ※ 주의: 마지막(stop) 숫자는 포함되지 않습니다!

4. 반복문 제어: break & continue

break (중단)

반복문을 즉시 탈출하고 종료합니다.

while True:
    n = int(input())
    if n == 0: break
    print(n)

continue (건너뛰기)

남은 코드를 무시하고 다음 반복으로 넘어갑니다.

for i in range(1, 6):
    if i == 3: continue
    print(i) # 3만 쏙 빠짐!

5. 반복문의 꽃: 중첩 반복문

반복문 안에 또 다른 반복문이 있는 형태입니다. 바깥쪽 반복문이 한 번 돌 때, 안쪽 반복문은 전체 바퀴를 다 돕니다.

실전 예제: 구구단 2단~3단 출력 
for i in range(2, 4):  # 바깥쪽 (단)
    print(f"--- {i}단 ---")
    for j in range(1, 4):  # 안쪽 (곱하는 수)
        print(i, "*", j, "=", i * j)
Execution Result 
--- 2단 ---
2 * 1 = 2
2 * 2 = 4
2 * 3 = 6
--- 3단 ---
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9

Challenge: 반복 + 선택 구조 믹스!

1부터 10까지 숫자 중 짝수만 합하는 코드를 작성해 볼까요?

sum_val = 0
for i in range(1, 11):
    if i % 2 == 0: # 짝수라면?
        sum_val += i
print("짝수 합:", sum_val)

  • • continue: 이번 차례 패스
  • • 중첩: 반복 속의 반복 (패턴 생성)
