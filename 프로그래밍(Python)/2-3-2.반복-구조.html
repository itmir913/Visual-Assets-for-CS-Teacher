Chapter 2. 프로그래밍의 기초

2-3-2. 반복 구조

같은 코드를 여러 번 작성할 필요 없어요!
원하는 만큼, 혹은 조건이 만족될 때까지 파이썬이 대신 반복하게 만드는 마법 같은 구조를 배워봅니다.

Think & Objective

"어떤 기준값이나 상태에 따라서 선택적으로 실행하거나 반복하는 구조는 어떻게 만들 수 있을까?"

  • 반복 구조를 사용할 수 있다.
  • 반복 구조의 중첩을 활용하여 복잡한 논리를 표현할 수 있다.

while 반복 구조

조건이 참(True)인 동안 명령을 반복해서 실행합니다. 조건이 언제 거짓이 될지 모르는 상황이나, 특정 상태가 유지되는 동안 반복할 때 유리합니다.

핵심 문법: 무한 루프(Infinite Loop) 방지

while문은 조건이 거짓이 되지 않으면 영원히 멈추지 않습니다.

  • 반복문 내부에서 조건을 변화시키는 코드(예: i += 1)가 반드시 포함되어야 합니다.
  • 의도적으로 무한 루프를 만들 때는 while True:를 사용합니다.

기본 형식 

while 조건식:
    반복할 코드
    조건을 변화시키는 코드 (무한루프 방지!)

1. while 문 : 조건이 참인 동안 반복

while 문은 주어진 조건이 True인 동안, 블록 안의 코드를 계속해서 반복 실행합니다. 반복이 끝나는 조건을 잘 만들어주는 것이 중요합니다!

다양한 실습 예제

예제 while문 기본 (1부터 3까지 출력)

변수 `n`이 1부터 시작하여 3 이하일 때까지 블록을 반복 실행합니다.

109.py 
n = 1
while n <= 3:
  print(n)
  n = n + 1
예제 입력받은 수까지 순차적으로 출력하기

사용자에게 숫자 `n`을 입력받고, `i`가 1부터 `n`이 될 때까지 반복하며 출력합니다.

110.py 
n = int(input())

i = 1
while i <= n:
  print(i)
  i = i + 1
예제 입력받은 수만큼 가로로 별(*) 출력하기

end="" 옵션을 사용하면 출력을 마친 후 줄바꿈을 하지 않고 이어서 출력할 수 있습니다.

111.py 
n = int(input())

i = 1
while i <= n:
  print("*", end="")
  i = i + 1

2. for 문과 range() : 횟수 지정 반복

몇 번 반복할지 명확할 때는 for 문이 편리합니다. 특히 숫자의 범위를 만들어주는 range() 함수와 단짝처럼 쓰입니다.

range(끝)

0부터 '끝-1'까지

예: range(3) → 0, 1, 2

range(시작, 끝)

시작부터 '끝-1'까지

예: range(1, 4) → 1, 2, 3

range(시작, 끝, 간격)

시작부터 지정한 간격만큼

예: range(0, 5, 2) → 0, 2, 4

기본 형식 

for 반복변수 in range(시작, 끝):
    반복코드
왜 for를 쓸까요?

for문은 반복 횟수가 명확하게 정해져 있을 때 사용합니다. range() 함수와 함께 사용하면 별도의 증감식 없이도 변수의 값을 자동으로 변화시킬 수 있어 편리합니다.

다양한 실습 예제

예제 for문 기본 (range 활용)

range(3)은 0부터 2까지 총 3번 반복을 의미합니다.

112.py 
for i in range(3):
  print(i)
예제 입력받은 수까지 for문으로 출력하기

1부터 입력한 숫자 n까지 포함하기 위해 range(1, n+1)로 범위를 지정합니다.

113.py 
n = int(input())

for i in range(1, n+1):
  print(i)

3. 리스트와 for 문 : 시퀀스 순회

for 문은 range() 뿐만 아니라 리스트, 문자열 같은 데이터 묶음(시퀀스)과도 아주 잘 맞습니다. 리스트 안에 있는 요소들을 처음부터 끝까지 하나씩 꺼내며 반복할 수 있습니다.

다양한 실습 예제

예제 문자열 리스트 순회하여 출력하기

공백을 기준으로 입력받아 분리된 단어들의 리스트 t에서 단어 s를 하나씩 순서대로 꺼내 출력합니다.

114.py 
t = input().split()

for s in t:
  print(s)
예제 리스트 역순으로 출력하기 (인덱스 활용)

range(n-1, -1, -1)을 사용하여 마지막 인덱스부터 0번 인덱스까지 거꾸로 접근합니다. 간격이 -1이므로 숫자가 감소합니다.

115.py 
d = list(map(int, input().split()))
n = len(d)
for i in range(n-1, -1, -1):
  print(d[i])
예제 여러 번 입력받아 리스트에 저장 후 출력하기

반복문을 통해 입력받은 값을 빈 리스트에 계속 append()(추가)하고, 다시 반복문을 돌며 한 줄로 출력합니다.

116.py 
n = int(input())

d = []
for i in range(n):
  d.append(int(input()))

for i in range(n):
  print(d[i], end=" ")

4. 반복문 제어 : break와 continue

때로는 반복을 도중에 멈추거나, 특정 상황에서만 건너뛰고 싶을 때가 있습니다.

break

만나는 즉시 반복문을 완전히 탈출합니다.

continue

아래 코드를 실행하지 않고, 다음 반복 단계로 넘어갑니다.

break

즉시 탈출

반복문 자체를 완전히 종료합니다.

for i in range(1, 11):
    if i == 7:
        break # 7이 되면 멈춤!
    print(i)
continue

이번만 건너뛰기

남은 코드를 무시하고 다음 주기로 넘어갑니다.

for i in range(1, 6):
    if i == 3:
        continue # 3만 빼고 출력!
    print(i)

다양한 실습 예제

예제 continue를 활용하여 특정 조건 건너뛰기

1부터 입력한 숫자까지 출력하되, 숫자가 3의 배수(i%3)==0일 경우 continue를 만나 아래의 print 코드를 실행하지 않고 건너뜁니다.

117.py 
n = int(input())

for i in range(1, n+1):
  if (i%3) == 0:
    continue
  print(i, end=" ")

5. 심화: 중첩 반복문과 알고리즘

반복문 안에 또 다른 반복문이 들어가는 형태를 중첩 반복문(Nested Loop)이라고 합니다. 다차원 데이터를 다루거나 복잡한 패턴(별 찍기), 그리고 특정 수학적 알고리즘(소수 판별 등)을 구현할 때 필수적으로 사용됩니다.

동작 원리 이해하기

"바깥쪽 반복문이 한 번 돌 때, 안쪽 반복문을 전부 실행합니다."

구구단 예시: 2단(바깥) 시작 → 1~9곱하기(안쪽 9번) 종료 → 3단(바깥) 시작...

다양한 실습 예제

예제 무한 루프와 break 응용

while True:를 통해 무한 반복을 만들고, 음수가 입력되면 break로 빠져나옵니다.

118.py 
while True:
  n = int(input())
  if n < 0:
    break
  print(n)
예제 중첩 while문으로 별(*) 삼각형 찍기

바깥쪽 반복문(`i`)은 줄 수를, 안쪽 반복문(`j`)은 해당 줄에서 출력할 별의 개수를 담당합니다.

119.py 
n = int(input())

i = 1
while i <= n:
  j = 1
  while j <= i:
    print("*", end="")
    j = j + 1
  print("")
  i = i + 1
예제 중첩 for문을 이용한 2차원 배열 출력

n행 m열의 격자 형태로 1부터 차례대로 숫자를 증가시키며 출력합니다.

120.py 
n = int(input())
m = int(input())

for i in range(n):
  for j in range(m):
    print((i*m) + j + 1, end=" ")
  print("")
예제 소수(Prime number) 판별하기 기본

2부터 n-1까지 나누어보며, 나누어 떨어지는 수가 있는지 확인합니다.

123.py 
n = int(input())

i = 2
while i < n:
  if (n % i) == 0:
    break
  i = i + 1

if i == n:
  print("prime")
else:
  print("composite")
예제 소수 판별 알고리즘 최적화 (제곱근)

모든 수를 검사할 필요 없이 i*i <= n 까지만 검사하여 실행 속도를 훨씬 빠르게 최적화한 방법입니다.

124.py 
n = int(input())

i = 2
while (i * i) <= n:
  if (n % i) == 0:
    break
  i = i + 1

if (i * i) > n:
  print("prime")
else:
  print("composite")
예제 n까지의 소수들을 리스트에 저장하기

중첩 반복문과 배열을 결합한 끝판왕 예제입니다! 지금까지 찾은 소수 리스트(`d`)를 활용하여 새로운 소수를 판별합니다.

125.py 
n = int(input())
d = [2]
i = 3
while i * i <= n:
  j = 0
  while (d[j] * d[j]) <= i:
    if (i % d[j]) == 0:
      break
    j = j + 1
  if (d[j] * d[j]) > i:
    d.append(i)
  i = i + 1
print(d)

반복 구조 핵심 요약

while (조건 중심)
for (횟수/데이터 중심)
break / continue (흐름 제어)
