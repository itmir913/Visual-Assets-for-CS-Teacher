1. 함수, 도대체 왜 쓰는 걸까?

똑같은 계산 과정을 코드 여기저기서 반복해서 적고 있다면, 여러분은 지금 함수가 절실히 필요한 상태입니다. 함수는 코드의 재사용성을 높이고 유지보수를 쉽게 만듭니다.

함수 사용 전 

# 원의 넓이를 3번 구해야 함
r1 = 5
area1 = 3.14 * r1 * r1
print(area1)

r2 = 10
area2 = 3.14 * r2 * r2
print(area2)

# 수식이 바뀌면 모든 곳을 고쳐야 함!

"코드가 길어지고 오타가 날 확률이 높아요!"

함수 사용 후 

def get_area(r):
    return 3.14 * r * r

print(get_area(5))
print(get_area(10))

# 이름만 부르면 끝! 수정도 한 곳만!

"가독성이 좋아지고 관리가 편해졌어요!"

2. 함수의 세 가지 종류

① 내장 함수 (Built-in)

파이썬을 설치하자마자 바로 쓸 수 있도록 준비된 함수입니다.

print() input() len() max()

② 라이브러리 함수 (Library)

특정 기능을 위해 모아둔 도구함에서 import로 꺼내 써야 합니다.

import math
print(math.sqrt(16)) # 제곱근 구하기

③ 사용자 정의 함수 (User-defined)

개발자가 필요에 의해 def 키워드로 직접 만든 함수입니다.

3. 함수 정의의 정석

문법 구조
def 함수이름(매개변수1, 매개변수2):
    # 수행할 문장
    return 결과값
자주 하는 실수: 들여쓰기

파이썬은 들여쓰기로 함수의 범위를 구분합니다. def 다음 줄부터는 반드시 Tab 또는 띄어쓰기 4칸을 맞춰야 합니다.

Return의 의미

return은 함수를 종료하고 값을 밖으로 던져줍니다. return이 실행되면 그 아래 코드는 무시됩니다!

4. 심화: 재귀 함수 (Recursion)

재귀 함수란 함수 안에서 **자기 자신을 다시 호출**하는 함수를 말합니다. 복잡한 문제를 단순하게 해결할 때 사용하지만, 자칫하면 무한 루프에 빠질 수 있어 주의가 필요합니다.

Example: 팩토리얼(!) 계산하기

def factorial(n):
    if n == 1: # 탈출 조건 (매우 중요!)
        return 1
    return n * factorial(n - 1) # 자신을 다시 호출

print(factorial(5)) # 결과: 120
⚠️ 재귀 함수의 핵심: 반드시 끝나는 조건(Base Case)이 있어야 합니다. 만약 n == 1일 때 멈추는 코드가 없다면 컴퓨터 메모리가 꽉 찰 때까지 계속 호출하다가 에러(RecursionError)가 발생합니다.

미션: 두 수의 평균 구하기

두 개의 숫자를 입력받아 평균을 리턴하는 함수 get_average를 만들고, 10과 20의 평균을 출력해 보세요. 

def get_average(a, b):
    result = (a + b) / 2
    return result

print(get_average(10, 20))
