1. 함수의 정의와 호출

Function Definition & Call

함수를 사용하는 과정은 크게 두 단계로 나뉩니다.
1. 정의하기(Define): "이런 일을 하는 함수를 만들겠어!"라고 선언하는 것
2. 호출하기(Call): "자, 이제 네 이름을 부를 테니 일을 시작해!"라고 명령하는 것

가장 기본적인 형태 (입력/출력 없음)

python
def say_hello(): # (1) 함수 정의
    print("안녕하세요!")
    print("함수 공부를 시작합니다.")

say_hello() # (2) 함수 호출 (이름을 불러야 실행됨)
📢 핵심 문법 체크:
  • def 예약어로 시작합니다.
  • 함수 이름 뒤에는 반드시 () 소괄호와 : 콜론을 붙입니다.
  • 함수에 속하는 문장들은 반드시 들여쓰기를 해야 합니다.

2. 매개변수: 함수에 값 전달하기

The Power of Parameters

함수가 특정 작업을 수행할 때 필요한 재료를 전달받는 통로가 바로 매개변수입니다.

입력값 (인수)
"철수"
함수의 문 (매개변수)
name
실행 결과
안녕, 철수!

Example: 매개변수 활용하기

def greet(name): # name이라는 매개변수를 받음
    print(f"안녕하세요, {name}님!")

greet("강해린") # "강해린"이 name에 전달됨
greet("하니")   # "하니"가 name에 전달됨
[연결학습] 우리가 이미 썼던 매개변수들

기본 제공 함수 print()에도 숨겨진 매개변수들이 있다는 사실, 기억나나요?

구분자 매개변수
print(a, b, sep=':')
끝문자 매개변수
print(a, end=' ')

3. 반환값: 결과 돌려받기

The Power of Return

함수가 일을 마친 후, 호출한 사람에게 계산 결과나 데이터를 다시 돌려주는 것을 반환(Return)이라고 합니다.

Example: 두 수의 합을 계산하는 함수

def add_numbers(a, b):
    sum_value = a + b
    return sum_value # 결과값을 밖으로 던집니다!

# 돌려받은 값을 변수에 저장할 수 있습니다.
result = add_numbers(10, 20)
print(result) # 출력: 30
학생들이 가장 많이 틀리는 포인트!
❌ 오답: return 뒤에 코드를 적는 경우
def test():
    return 10
    print("이건 실행 안 됨")

반환문(return)이 실행되면 함수는 그 즉시 즉시 종료됩니다!

❌ 오답: return이 없는데 값을 받으려는 경우
def no_return(a):
    print(a)

val = no_return(5)
print(val) # None이 출력됩니다!

결과값을 돌려주지 않으면 변수에는 아무것도 남지 않아요.

오늘의 설계 챌린지 🛠️

배운 내용을 모두 활용해 '성적 계산기'를 만들어 봅시다.

설계 미션
  • 1. calc_grade(score) 함수를 만듭니다.
  • 2. 점수가 80 이상이면 "Pass", 아니면 "Fail"을 리턴합니다.
  • 3. 사용자에게 점수를 입력받아 결과를 출력하세요.
def calc_grade(score):
    if score >= 80:
        return "Pass"
    else:
        return "Fail"

s = int(input("점수 입력: "))
result = calc_grade(s)
print(f"당신의 결과는 {result}입니다.")
