영구적인 데이터 저장의 기본

휘발되는 데이터는 이제 안녕!
데이터를 영구적으로 저장하는 방법을 알아봅시다.

1. 왜 파일이 필요할까요?

Memory vs Storage

변수 (메모리)

프로그램을 끄면 사라집니다. 마치 우리가 공부할 때 잠깐 머릿속에 담아두는 것과 같죠.

파일 (저장장치)

프로그램을 꺼도 남아있습니다. 공부한 내용을 공책에 적어 보관하는 것과 같습니다.

컴퓨터 파일(File)은 일정한 규칙에 따라 저장 장치에 기록한 데이터의 집합입니다. 파이썬은 이 파일을 읽고 쓰는 아주 쉬운 방법을 제공합니다.

2. 파일 입출력 과정

The 3-Step Process

파일을 다루는 것은 '책을 읽는 과정'과 똑같습니다. 순서를 꼭 기억하세요!

1
파일 열기

책장에 가서 책을 꺼내 펴기

open()
2
읽기 또는 쓰기

책 내용을 읽거나 필기하기

read() / write()
3
파일 닫기

책을 덮고 책장에 넣기

close()

가장 기본적인 파일 쓰기 예제

python
# 1. 파일 열기 (이름: hello.txt, 모드: 'w'rite)
f = open("hello.txt", "w")

# 2. 파일에 내용 쓰기
f.write("반갑다, 파일 처리는 처음이지?")

# 3. 파일 닫기 (반드시 닫아야 저장됩니다!)
f.close()

3. 파일 열기 모드

Mode Selection

파일을 열 때 "어떤 목적으로 열 것인가"를 정해줘야 합니다. 잘못 선택하면 데이터가 지워질 수 있으니 주의하세요!

'r'
읽기 (Read)

파일 내용을 읽기만 할 때 사용합니다. 파일이 없으면 에러가 나요.

'w'
쓰기 (Write)

새로 작성할 때 사용합니다. 기존 내용은 모두 사라지니 주의!

'a'
추가 (Append)

기존 내용 뒤에 이어서 쓰고 싶을 때 사용합니다.

주의하세요! 'w'의 함정

open("data.txt", "w")를 실행하는 순간, 만약 data.txt가 이미 존재했다면 그 안의 내용은 깨끗하게 삭제되고 빈 파일이 됩니다. 내용을 보존하면서 뒤에 덧붙이고 싶다면 반드시 'a' 모드를 사용하세요.

4. 파일 내용 가져오기

Reading the Data

파일에 저장된 데이터를 프로그램 안으로 불러오는 코드입니다.

# 1. 읽기 모드로 파일 열기
f = open("hello.txt", "r")

# 2. 내용 전체를 읽어 변수에 저장하기
data = f.read()
print(data)

# 3. 닫기
f.close()
🤔 "선생님, f.close()를 꼭 해야 하나요?"

파일을 열어두고 닫지 않으면, 컴퓨터는 그 파일을 "사용 중"이라고 생각해서 다른 프로그램이 수정하지 못하게 잠가버릴 수 있어요. 또한, 닫지 않고 프로그램이 종료되면 쓴 내용이 저장이 안 될 수도 있답니다. 열었으면 반드시 닫는 습관! 잊지 마세요.
