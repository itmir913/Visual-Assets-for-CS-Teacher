1. TXT 파일에 저장하기

Writing to Text Files

파이썬에서 문자열을 저장할 때는 f.write()를 사용합니다. 줄을 바꾸고 싶다면 "\n"을 반드시 붙여주세요.

예제: hello, world 저장하기

python
f = open("out.txt", "w")
f.write("hello\n")
f.write("python\n")
f.write("world\n")
f.close()
1부터 n까지 제곱수 저장하기 (Tip)

숫자를 파일에 저장할 때는 반드시 str() 함수를 사용해 문자열로 변환해야 합니다!
f.write(str(i*i)) 처럼 작성해 보세요.

2. TXT 파일 데이터 불러오기

Reading from Text Files

f.read()

파일의 전체 내용을 하나의 문자열로 가져옵니다.

.split("\n")

줄바꿈을 기준으로 문자열을 잘라 리스트로 만듭니다.

예제: 대문자로 변환하여 출력하기 

f = open("in.txt", "r")
rows = (f.read()).split("\n")
f.close()

# 리스트를 돌며 대문자로 출력
for row in rows:
    print(row.upper())

3. CSV 파일 다루기

Comma Separated Values

CSV 파일이란?

데이터 사이를 쉼표(,)로 구분하여 저장하는 텍스트 파일입니다. 엑셀과 같은 스프레드시트 프로그램과 호환이 매우 잘 됩니다.

예제: 줄별로 합 계산하기 

# 한글 깨짐 방지를 위해 encoding 설정
f = open("datax.csv", "r", encoding="utf-8-sig")
rows = (f.read()).split("\n")
f.close()

for row in rows:
    values = row.split(",")
    s = 0
    for value in values:
        s = s + int(value)
    print(s)

4. 데이터를 CSV로 저장하기

Exporting Data to CSV

리스트 형태의 데이터를 쉼표와 줄바꿈을 조합하여 파일로 만듭니다.

rows = [["num", "data1"], [1, 79], [2, 83]]
f = open("dataoutx.csv", "w")

for row in rows:
    # 마지막 요소 전까지 쉼표로 연결
    for value in row[:-1]:
        f.write(str(value) + ",")
    # 마지막 요소 쓰고 줄바꿈
    f.write(str(row[-1]) + "\n")
f.close()
