원의 겹침 여부 판별하기

두 원의 좌표와 반지름을 이용하여 논리적으로 위치 관계를 계산하는 프로그램

Step 1. 프로그램 기획 (문제 정의)

두 원의 중심 좌표 $(x1, y1), (x2, y2)$와 각 원의 반지름 $r1, r2$를 입력받아, 두 원이 어떤 상태에 있는지 판별하는 프로그램을 개발합니다.

판별 케이스 1

두 원이 서로 다른 두 점에서 만나는 경우

판별 케이스 2

두 원이 한 점에서 만나는 경우 (외접/내접)

판별 케이스 3

두 원이 만나지 않는 경우

Step 2. 프로그램 설계 (알고리즘)

핵심 알고리즘은 '두 점 사이의 거리 공식'과 '반지름의 합/차'를 비교하는 것입니다.

두 중심 사이의 거리 (d) =
$\sqrt{(x2 - x1)^2 + (y2 - y1)^2}$

판별 로직 설계

  • 상태 1: $d > r1 + r2 \rightarrow$ 만나지 않음
  • 상태 2: $d = r1 + r2 \rightarrow$ 한 점에서 만남 (외접)
  • 상태 3: $d < r1 + r2 \rightarrow$ 두 점에서 만남 (단, 큰 원 안에 작은 원이 없는 조건)

Step 3. 프로그램 구현 (Python)

math 라이브러리의 sqrt 함수와 조건문을 활용하여 구현합니다.

import math

# 두 원의 정보 입력
x1, y1, r1 = map(int, input().split())
x2, y2, r2 = map(int, input().split())

# 두 중심 사이의 거리 계산
d = math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

# 위치 관계 판별
if d > r1 + r2:
    print("만나지 않는다")
elif d == r1 + r2:
    print("한 점에서 만난다")
else:
    print("두 점에서 만난다")
