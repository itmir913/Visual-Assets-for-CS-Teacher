순위 데이터 정렬: 3등 찾기

이름과 점수가 한 쌍인 다차원 리스트를 정렬하여 특정 순위의 데이터를 추출합니다.

Step 1. 프로그램 기획 (문제 정의)

$n$명의 친구들의 이름과 정보 성적이 주어집니다. 이 데이터들 중에서 성적이 세 번째로 높은 학생을 찾아 그 이름을 출력하는 프로그램을 기획합니다.

2등
이민수(95)
1등
김철수(98)
3등
박영희(92)

데이터 예시: [ ['김철수', 98], ['이민수', 95], ['박영희', 92], ... ]

Step 2. 프로그램 설계 (알고리즘)

1. 데이터 구조화

이름(String)과 점수(Int)를 하나의 묶음으로 처리하기 위해 2차원 리스트를 활용합니다.

2. 정렬 기준 설정

리스트의 1번 인덱스(점수)를 기준으로 내림차순 정렬을 수행해야 합니다.

핵심 아이디어: 파이썬의 sort() 함수와 lambda 식을 이용하면 다차원 리스트를 특정 열 기준으로 쉽게 정렬할 수 있습니다.

Step 3. 프로그램 구현 (Python)

# n명의 정보 입력 받기
n = int(input())
students = []

for _ in range(n):
    name, score = input().split()
    students.append([name, int(score)])

# 점수(index 1) 기준 내림차순 정렬
students.sort(key=lambda x: x[1], reverse=True)

# 3등(index 2) 학생의 이름 출력
print(students[2][0])

Step 4. 디버깅 및 시스템 테스트

디버깅 포인트

데이터 입력 시 점수가 문자열로 저장되지 않도록 int() 형변환을 거쳤는지 확인합니다.
