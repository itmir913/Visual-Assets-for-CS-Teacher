교집합과 합집합 계산기

집합(Set) 자료구조를 활용하여 두 집합의 공통 원소와 전체 원소를 오름차순으로 추출합니다.

Step 1. 프로그램 기획 (문제 정의)

두 개의 숫자 집합을 입력받아 수학의 집합 연산을 수행합니다. 결과는 반드시 오름차순으로 정렬하여 출력해야 합니다.

핵심 개념: 집합(Set) 자료구조

중복을 허용하지 않으며 순서가 없는 데이터 묶음입니다.

Set A
Set B
교집합 (A ∩ B)

교집합(Intersection)

두 집합에 공통으로 포함된 원소들을 추출합니다. 파이썬 연산자: &

합집합(Union)

두 집합의 모든 원소를 합칩니다. (중복은 자동 제거) 파이썬 연산자: |

Step 2. 프로그램 설계 (알고리즘)

입력된 데이터를 집합으로 변환하고 정렬하는 과정을 설계합니다.

알고리즘 순서 설계
  1. 두 집합의 원소 개수와 데이터를 각각 입력받음
  2. 입력받은 리스트를 set()으로 변환
  3. 집합 연산(&, |) 수행
  4. 결과 집합을 sorted() 함수를 통해 리스트로 변환 및 정렬
  5. 결과가 비어있는지(공집합) 확인 후 출력

Step 3. 프로그램 구현 (Python)

집합 자료구조와 내장 함수를 활용한 구현 예시입니다.

# 입력 및 집합 변환
set_a = set(map(int, input().split()))
set_b = set(map(int, input().split()))

# 교집합 연산 및 정렬
inter = sorted(list(set_a & set_b))
if not inter:
    print(0) # 공집합 처리
else:
    print(*inter)

# 합집합 연산 및 정렬
uni = sorted(list(set_a | set_b))
print(*uni)
주의사항

집합(set)은 자체적으로 순서가 없으므로, 출력 시 반드시 sorted()를 사용하여 오름차순 정렬 과정을 거쳐야 채점 시스템에서 정답으로 인정됩니다.

Step 4. 프로그램 테스트 및 검토

검증 케이스

1. 두 집합이 완전히 겹칠 때
2. 두 집합에 공통 원소가 없을 때(공집합)
3. 한 집합이 다른 집합의 부분집합일 때

온라인 채점 확인

알고리즘이 수초 이내에 종료되는지 확인하고 시스템 테스트를 수행합니다.
