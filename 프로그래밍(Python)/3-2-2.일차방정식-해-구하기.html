일차방정식 \(ax \pm b = c\) 해 구하기

문자열 분리 기술을 활용하여 수학 문제를 해결하는 똑똑한 프로그램을 만들어 봅시다.

수학과 코딩의 만남

우리는 수학 시간에 일차방정식을 푸는 방법을 배웠습니다. 이번 프로젝트의 핵심은 **사용자가 입력한 문장(문자열)에서 숫자만 쏙쏙 뽑아내어(Parsing)** 컴퓨터가 계산하게 만드는 것입니다.

Step 1. 프로그램 기획 (요구 사항)

사용자가 일차방정식 형태의 식을 입력하면 컴퓨터가 \(x\)의 값을 계산해 주어야 합니다.

\(ax + b = c\)   또는   \(ax - b = c\)

입력 조건

공백을 포함한 문자열 형태로 식이 입력됩니다.
예: "3x + 5 = 20"

출력 조건

계산된 \(x\)의 값을 출력합니다. (나누어떨어지지 않을 경우 소수점 처리 고려)

Step 2. 프로그램 설계 (해법 찾기)

일차방정식을 풀기 위해 우리가 손으로 푸는 과정을 알고리즘으로 정리해 봅시다.

해 구하기 공식 (이항 정리)

1. \(b\)를 우변으로 넘긴다: \(ax = c - b\) (혹은 \(c + b\))

2. \(a\)로 양변을 나눈다: \(x = (c - b) / a\)

문자열 분리(Split) 전략

입력된 "3x + 5 = 20"에서 숫자 3, 5, 20을 어떻게 분리할까요? 파이썬의 split() 함수와 문자열 슬라이싱(Slicing)을 적절히 혼합해야 합니다.

Step 3. 프로그램 구현 (Source Code)

가장 범용적으로 사용할 수 있는 코드를 작성해 봅시다.

Python Solution
# 1. 입력 받기 (예: 3x + 5 = 20)
equation = input("방정식을 입력하세요 (예: 3x + 5 = 20): ")

# 2. 문자열 분리하기 (공백 기준)
parts = equation.split()
# parts[0] -> "3x", parts[1] -> "+", parts[2] -> "5", parts[4] -> "20"

# 3. 데이터 가공 (Parsing)
a = float(parts[0][:-1]) # "3x"에서 마지막 'x' 제외하고 숫자로
op = parts[1]                # "+" 또는 "-"
b = float(parts[2])
c = float(parts[4])

if op == '+':
    x = (c - b) / a
else:
    x = (c + b) / a

print(f"x의 값은 {x:.2f}입니다.")
핵심 문법 설명: parts[0][:-1]

리스트 parts[0]에 담긴 "3x"라는 글자에서 마지막 "x"를 떼어내야 숫자로 변환할 수 있습니다. 이때 슬라이싱 [:-1]을 사용하면 "처음부터 마지막 글자 바로 전까지"만 잘라낼 수 있어 매우 편리합니다!

Step 4. 테스트 및 예외 상황

프로그램이 멈추지 않도록 학생들이 자주 실수하는 부분을 점검합히다.

이런 경우를 조심하세요!
  • 계수 \(a\)가 1인 경우: 사용자가 "x + 5 = 10"이라고 입력하면 parts[0]은 그냥 "x"가 됩니다. 이때 float(parts[0][:-1])을 하면 빈 문자열 오류가 납니다. (조건문으로 \(a=1\) 처리가 필요합니다.)
  • 공백 유무: split()은 공백을 기준으로 나누기 때문에, 사용자가 "3x+5=20"처럼 공백 없이 입력하면 프로그램이 작동하지 않습니다. "공백을 지켜주세요"라는 안내가 기획 단계에서 중요합니다.
