리스트와 조건문을 활용하여 특정 연도와 월의 마지막 날을 계산해 봅시다.

프로젝트 배경

매달 말일은 30일이기도 하고 31일이기도 합니다. 특히 **2월은 윤년 여부에 따라 28일 또는 29일**이 됩니다. 규칙적인 데이터를 효율적으로 관리하는 **리스트(List)** 기능을 사용하여 이 문제를 해결해 봅시다.

Step 1. 프로그램 기획 (요구 사항)

사용자로부터 연도와 월을 입력받아, 해당 월의 마지막 날짜를 출력합니다.

입력 예시

공백을 구분자로 연도와 월이 입력됩니다.
예: 2024 2

출력 예시

해당 월의 마지막 날짜 숫자를 출력합니다.
예: 29

Step 2. 프로그램 설계 (핵심 로직)

데이터 관리와 조건 처리를 위한 두 가지 핵심 전략을 세웁니다.

1) 리스트로 데이터 관리하기

각 월의 마지막 날짜를 리스트에 담아두면 인덱스 번호로 쉽게 접근할 수 있습니다.

31
[0]
28

31
[2]
30
[3]
...
31
[11]
2) 윤년(Leap Year) 판별 조건식

2월의 말일을 결정하는 윤년의 규칙은 다음과 같습니다.

  • 연도가 400의 배수이면 윤년입니다.
  • 또는, 연도가 4의 배수이면서 100의 배수가 아니면 윤년입니다.

\((year \% 400 == 0) \text{ or } (year \% 4 == 0 \text{ and } year \% 100 \neq 0)\)

Step 3. 프로그램 구현 (Source Code)

Python Solution
# 1. 입력 받기 (연도와 월)
year, month = map(int, input().split())

# 2. 각 달의 마지막 날 리스트 (1월 ~ 12월)
days = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

# 3. 윤년인 경우 2월(인덱스 1)의 날짜를 29로 변경
if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
    days[1] = 29

# 4. 결과 출력 (입력받은 month에 해당하는 날짜)
# 인덱스는 0부터 시작하므로 month-1을 해줘야 함
print(days[month - 1])
친절한 문법 설명: 인덱스(Index)의 함정

우리는 보통 1월을 '첫 번째 달'이라고 부르지만, 파이썬 리스트의 첫 번째 칸 번호는 **0**입니다. 따라서 사용자가 입력한 month에서 1을 빼주어야 정확한 데이터를 가져올 수 있습니다.
