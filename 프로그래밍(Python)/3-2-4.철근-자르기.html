자재 자동 라벨링: 철근 자르기

반복문과 연산자를 활용하여 자재 번호를 자동으로 생성하는 시스템을 설계합니다.

현장의 문제를 코딩으로!

공사 현장에서 긴 철근을 일정한 길이로 자른 뒤, 각각에 **식별 번호(F-0001, F-0002...)**를 붙여야 한다고 상상해 보세요. 손으로 쓰기엔 너무 많죠? 컴퓨터를 이용해 **자재 번호를 자동으로 생성**하는 프로그램을 만들어 봅시다.

Step 1. 프로그램 기획 (요구 사항)

입력된 전체 길이와 자를 길이를 바탕으로 생성 가능한 모든 자재 번호를 출력합니다.

[철근 커팅 예시]

전체 길이 (100)
F-0001
F-0002
F-0003
... (33개 생성)

남은 자투리(길이 1)는 번호를 부여하지 않고 제외합니다.

입력 조건

전체 길이 \(L\)과 자를 길이 \(k\)가 공백으로 입력됩니다.
예: 100 3

출력 조건

F-0001부터 시작하여 1씩 증가하며 번호를 한 줄에 하나씩 출력합니다.

Step 2. 프로그램 설계 (해법)

1) 몇 개를 만들 수 있을까? (산술 연산)

전체 길이를 자를 길이로 나눈 **몫**만큼 자재 번호를 생성할 수 있습니다.

\(count = total // cut\)

2) 형식에 맞게 출력하기 (포맷팅)

단순히 1, 2, 3이 아니라 0001, 0002처럼 4자리를 맞추는 것이 포인트입니다.

  • 파이썬 f-string 활용: f"F-{i:04d}"
  • :04d는 4자리 정수로 출력하되 빈 곳은 0으로 채우라는 뜻입니다.

Step 3. 프로그램 구현 (Source Code)

Python Solution
# 1. 데이터 입력 (전체 길이, 자를 길이)
total, cut = map(int, input().split())

# 2. 생성 가능한 자재 개수 계산
count = total // cut

# 3. 반복문을 이용한 번호 생성 및 출력
for i in range(1, count + 1):
    # f-string의 :04d 포맷을 사용해 0001 형식 구현
    print(f"F-{i:04d}")
반복문 range(1, count + 1) 이해하기

우리는 1번 자재부터 필요하므로 시작값을 1로 잡습니다. 끝값은 포함되지 않기 때문에 count + 1까지 범위를 지정해야 마지막 번호까지 출력됩니다.
