확률로 푸는 수학: 파이(π)값 근사하기

무작위 좌푯값을 이용하여 원주율을 계산하는 시뮬레이션 프로그램을 제작합니다.

다트 던지기로 π 찾기?

한 변의 길이가 1인 정사각형 안에 무작위로 점을 찍는다고 상상해 보세요. 그중 반지름이 1인 사분원(부채꼴) 안에 들어가는 점의 비율은 얼마나 될까요? 이 **기하학적 확률**을 이용하면 신비로운 숫자 π를 프로그래밍으로 찾아낼 수 있습니다.

Step 1. 프로그램 기획 (요구 사항)

무작위로 생성된 \(n\)개의 좌푯값이 주어졌을 때, 공식에 따라 \(\pi\)의 근삿값을 계산합니다.

빨간 점: 원 내부 / 파란 점: 원 외부

입력 데이터

전체 점의 개수 \(n\)과 각 점의 \((x, y)\) 좌표 리스트.
예: [[0.5, 0.5], [0.9, 0.8], ...]

출력 결과

계산된 \(\pi\)의 근삿값을 소수점 아래까지 출력합니다.

Step 2. 프로그램 설계 (핵심 공식)

몬테카를로 알고리즘의 핵심 로직을 설계합니다.

원 안에 있는지 판별하는 법

점 \((x, y)\)와 원점 \((0, 0)\) 사이의 거리가 반지름 \(1\)보다 작거나 같으면 원 내부에 있습니다.

$$x^2 + y^2 \leq 1$$

π 근사 공식

정사각형 면적에 대한 사분원 면적의 비율은 \(\pi/4\)입니다. 따라서:

$$\pi \approx 4 \times \frac{\text{원 안의 점 개수}}{\text{전체 점의 개수}}$$

Step 3. 프로그램 구현 (Source Code)

Python Solution
# 1. 테스트용 좌표 데이터 (2차원 리스트)
points = [[0.1, 0.2], [0.9, 0.8], [0.3, 0.4], [0.5, 0.5]]
n = len(points)
in_count = 0 # 원 안에 들어간 점의 개수

# 2. 각 좌표를 순회하며 원 안에 있는지 검사
for p in points:
    x = p[0]
    y = p[1]
    # 거리의 제곱이 1 이하면 원 안임
    if (x**2 + y**2) <= 1:
        in_count += 1

# 3. 파이값 계산 및 출력
pi_approx = 4 * (in_count / n)
print("파이 근삿값:", pi_approx)
2차원 리스트 다루기

points[i][0]은 \(i\)번째 점의 x좌표, points[i][1]은 y좌표를 의미합니다. 리스트의 인덱스 구조를 정확히 파악하는 것이 중요합니다!

Step 4. 프로그램 테스트 및 고도화

결과의 정확성

점의 개수가 많아질수록(예: 10,000개) \(\pi\) 값인 3.14159...에 점점 더 가까워집니다.

랜덤 모듈 활용

직접 좌표를 입력하는 대신 random.random() 함수를 써서 컴퓨터가 스스로 점을 찍게 해보세요.

생각해보기

왜 하필 4를 곱할까요? 원의 전체 면적 공식은 \(\pi r^2\)이고, 반지름이 1인 원의 면적은 \(\pi\)입니다. 우리가 계산한 사분원은 원의 1/4이므로 \(\pi/4\)가 나오기 때문입니다.
