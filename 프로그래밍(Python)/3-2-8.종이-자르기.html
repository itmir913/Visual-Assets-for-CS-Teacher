종이 자르기: 격자 프로그래밍

중첩 반복문을 사용하여 가로와 세로 규격에 맞는 점선 패턴을 출력합니다.

점선 그리기 프로젝트

고객이 원하는 가로(W)와 세로(H) 길이에 맞춰 큰 종이에 자르는 선을 표시해야 합니다. 컴퓨터는 위에서 아래로, 왼쪽에서 오른쪽으로 한 줄씩 출력하므로, 우리는 **행(Row)**과 **열(Column)**의 개념을 반복문으로 구현해야 합니다.

Step 1. 프로그램 기획 (요구 사항)

사용자로부터 가로와 세로 길이를 입력받아 지정된 형식의 점선을 출력합니다.

[출력 예시: 가로 4, 세로 3]

+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+

입력 조건

두 개의 정수 \(W\)(가로), \(H\)(세로)가 공백을 두고 입력됩니다.
예: 4 3

출력 조건

가로 \(W\)칸, 세로 \(H\)칸의 격자 모양을 기호 +, -, |를 조합하여 출력합니다.

Step 2. 프로그램 설계 (구조 분석)

격자는 크게 두 종류의 줄이 번갈아 가며 나옵니다.

1. 가로 경계선 (Border Line)

패턴: +---가 \(W\)번 반복되고 마지막에 +가 붙음.

예시: +---+---+---+---+

2. 세로 칸 구분선 (Content Line)

패턴: | (공백 3칸)이 \(W\)번 반복되고 마지막에 |가 붙음.

예시: | | | | |

중첩 반복문의 구조

전체 세로 길이 \(H\)만큼 반복하면서, 그 안에서 가로 길이 \(W\)만큼 기호를 출력해야 합니다. 즉, 바깥쪽 루프는 '행(세로)', 안쪽 루프는 '열(가로)'을 담당합니다.

Step 3. 프로그램 구현 (Source Code)

Python Solution
# 1. 가로(w)와 세로(h) 입력 받기
w, h = map(int, input().split())

# 2. 전체 세로 횟수만큼 반복
for i in range(h):
    # 가로 경계선 출력
    print("+---" * w + "+")
    # 세로 칸 구분선 출력
    print("| " * w + "|")

# 3. 마지막 바닥 경계선 출력
print("+---" * w + "+")
파이썬 꿀팁: 문자열 곱셈

파이썬에서는 "문자열" * 숫자를 통해 반복적인 패턴을 아주 쉽게 만들 수 있습니다. 복잡한 안쪽 반복문 대신 이 기능을 활용하면 코드가 훨씬 간결해집니다!

Step 4. 프로그램 테스트 및 검증

테스트 데이터 1

입력: 1 1
출력: 가장 작은 1x1 격자가 정확히 나오는지 확인

테스트 데이터 2

입력: 10 2
출력: 옆으로 아주 긴 격자가 출력되는지 확인

심화 도전 과제

1. 2중 반복문(중첩 루프)만 사용하여 출력해 보세요. (문자열 곱셈 금지!)
2. 칸 안에 번호를 매겨 보세요 (예: 1, 2, 3...).
3. 사용자가 칸의 너비(---의 개수)도 조절하게 만들 수 있을까요?

격자 출력을 마스터했나요? 온라인 저지에서 실력을 검증하세요!
