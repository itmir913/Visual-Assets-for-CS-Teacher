복면산(覆面算) 문제 해결

글자로 숨겨진 숫자를 논리적 추론과 반복문으로 찾아내는 퍼즐 프로그램

Step 1. 프로그램 기획 (문제 정의)

복면산이란 각 문자가 서로 다른 숫자를 나타내는 수학 퍼즐입니다. 같은 문자는 같은 숫자를 의미하며, 수의 맨 앞자리(최고차항)는 0이 될 수 없다는 규칙이 있습니다.

  A B
+ B A
-----
  C C

위 문제에서 A, B, C가 각각 어떤 숫자일 때 식이 성립하는지 찾아내는 프로그램을 개발합니다.

Step 2. 프로그램 설계 (알고리즘)

핵심 로직: 브루트 포스 (Brute Force)

컴퓨터의 빠른 연산 능력을 활용하여 가능한 모든 숫자 조합(0~9)을 대입해보고 조건에 맞는 답을 찾습니다.

1. 범위 설정

A, B, C는 각각 0부터 9까지의 숫자가 될 수 있습니다.

2. 중복 제약

A, B, C는 서로 다른 숫자여야 합니다. (A != B and B != C and A != C)

3. 수식 검증

(10*A + B) + (10*B + A) == (11*C) 조건이 참인지 확인합니다.

Step 3. 프로그램 구현 (Python)

다중 for 문과 continue 문을 사용하여 효율적으로 구현합니다.

# 3중 반복문으로 모든 경우의 수 탐색
for a in range(1, 10): # A는 첫 자리이므로 1~9
    for b in range(1, 10): # B도 첫 자리이므로 1~9
        if a == b: continue # A와 B가 같으면 건너뜀

        for c in range(1, 10):
            if c == a or c == b: continue

            # 조건 확인: AB + BA == CC
            if (a * 10 + b) + (b * 10 + a) == (c * 10 + c):
                print(f"A={a}, B={b}, C={c}")
주의사항

반복문의 횟수가 늘어날수록 컴퓨터가 처리해야 할 양이 기하급수적으로 늘어납니다. continue를 적절히 사용해 불필요한 연산을 줄이는 것이 '최적화'의 핵심입니다.

Step 4. 프로그램 테스트 및 검토

실행 결과 확인

프로그램을 실행했을 때 조건에 맞는 결과가 출력되는지 확인합니다.

예상 출력: A=1, B=2, C=3 (12+21=33) 등 여러 쌍이 나올 수 있습니다.
